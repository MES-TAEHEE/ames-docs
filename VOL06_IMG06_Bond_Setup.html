01Overview개요
EN · Description Bond machine recipe parameter screen — sets pressure / temperature / time setpoints for the wrapping bond machine. Recipe is auto-loaded from MD_Recipe when a WO is accepted (IMG-03); operators can fine-tune within MD-defined tolerance. Real-time PLC feedback shown for current vs setpoint. Out-of-spec changes require supervisor PIN. Bond temperature history feeds QC for adhesion analysis.
KO · 설명 본드 기계 레시피 파라미터 화면 — 감싸기 본드기의 압력·온도·시간 설정값 관리. 레시피는 WO 수락(IMG-03) 시 MD_Recipe에서 자동 로드. 작업자가 MD 정의 허용오차 내에서 미세 조정 가능. PLC 실시간 피드백(현재값 vs 설정값) 표시. 허용오차 외 변경은 라인장 PIN 필요. 본드 온도 이력은 QC 접착력 분석에 활용.

03Recipe Logic레시피 로직
StageDetail
1. Auto LoadIMG-03 WO Accept 시 MD_Recipe[ItemNo].BondRecipeID로 레시피 자동 로드. PLC로 setpoint 다운로드.
2. PLC SyncOPC UA로 PLC 현재값 1초 주기 폴링. UI는 5초 캐시 갱신.
3. Within Tolerance작업자가 ±Tolerance 내 미세 조정 가능 (Supervisor PIN 불필요).
4. Out-of-Spec±Tolerance 초과 시 PIN 모달. 사유 코드(BOND_LOT_VARIATION / FABRIC_SUB / OPTIMIZATION) 필수.
5. Per-cycle Recording매 사이클 평균/최대/최소 압력·온도·시간을 BondCycleLog에 저장 (IMG-03 EntryID 연계).
6. Out-of-Spec Cycle Flag사이클 결과값이 tolerance 외 → QC 강화검사 자동 플래그 (BR-IMG-S03).
7. Reset[Reset to MD Default] 버튼: 모든 setpoint를 MD_Recipe 표준값으로 복원. PIN 필요.
04Workflow작업 흐름
IMG-03 WO Accept fires → IMG-06 auto-loads MD_Recipe + PLC setpoint download.
IMG-03 WO Accept → IMG-06이 MD_Recipe 자동 로드 + PLC 다운로드.
Operator opens IMG-06 to monitor current vs setpoint during runtime.
작업자가 IMG-06 열어 가동 중 현재값 vs 설정값 모니터링.
Within tolerance → operator adjusts directly. PLC updated immediately.
허용오차 내 조정 시 즉시 PLC 반영.
Out of tolerance → PIN modal + reason code → audit log + PLC apply.
허용오차 외 시 PIN 모달 + 사유 → 감사 기록 + PLC 적용.
Each cycle: PLC measured values logged + tolerance check + QC flag if needed.
매 사이클 PLC 실측값 기록 + tolerance 점검 + 필요 시 QC 플래그.
