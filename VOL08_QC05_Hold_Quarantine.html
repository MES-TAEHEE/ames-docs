01Overview개요
EN · Description The physical and system-level isolation gate for non-conforming materials and products. Auto-created by Major+ NCRs (QC-04) or directly by QC Lead. Each Hold (1) flags the affected stock/WO/LOT in DB so no downstream operation can ship/move/use it, (2) prints a red Quarantine label with HoldID barcode applied to the physical location/container, (3) writes the location to WH-04 quarantine map. Release requires QC Lead+ PIN entry plus a Disposition outcome from NCR (or an explicit override with Plant Manager approval). All hold/release events ≥ 5y retention.
KO · 설명 부적합 자재·제품의 물리·시스템 격리 게이트. Major 이상 NCR(QC-04)이 자동 생성하거나 QC Lead가 직접 발의. 각 Hold는 (1) 영향 재고/WO/LOT을 DB에서 플래그하여 하류 출고/이동/사용 차단, (2) HoldID 바코드 빨강 격리 라벨을 물리 위치/용기에 부착, (3) WH-04 격리 맵에 위치 기록. 해제는 QC Lead 이상 PIN + NCR Disposition 결과 필수 (긴급 override 시 공장장 승인). 모든 격리/해제 이벤트 5년 이상 보관.

04Workflow작업 흐름
Trigger: NCR(QC-04) ≥ Major auto-creates Hold OR QC Lead manually creates Hold for suspect material.
트리거: NCR Major 이상 자동 생성 또는 QC Lead가 의심 자재 직접 발의.
System updates affected stock/WO/LOT flag (DB constraint) → all downstream ops blocked instantly.
시스템이 영향 재고/WO/LOT 플래그 갱신 (DB 제약) → 하류 작업 즉시 차단.
Red Quarantine label auto-printed (HoldID barcode + warning); operator affixes to physical container.
빨강 격리 라벨 자동 인쇄(HoldID 바코드 + 경고), 작업자가 물리 컨테이너에 부착.
Material moved to Q-zone (WH-04 quarantine map); position recorded; PDA scan confirms placement.
자재를 Q-zone(WH-04 격리 맵)으로 이동, 위치 기록, PDA 스캔으로 배치 확인.
QC Lead reviews linked NCR, awaits Disposition decision (REWORK/SCRAP/UAI/RETURN).
QC Lead가 연관 NCR 검토, Disposition 결정 대기 (REWORK/SCRAP/UAI/RETURN).
On Disposition decided: Lead enters PIN + Reason + selects Release Action; CRITICAL needs Plant Mgr approval.
Disposition 결정 후: Lead가 PIN + 사유 + Release Action 선택, CRITICAL은 공장장 승인 추가.
Downstream cascade: RELEASE → Available stock, REWORK → new WO, SCRAP → inventory write-off + scrap LOT.
하류 연계: RELEASE → Available, REWORK → 신규 WO, SCRAP → 재고 차감 + 폐기 LOT.
All events (Hold/Release/Scrap) sealed for ≥ 5 years; visible in QC-TRC Traceability.
모든 이벤트 (Hold/Release/Scrap) 5년 이상 봉인, QC-TRC에서 조회 가능.

LOCKED 항목 이동/출하/소비 차단 (FG·WH·MRP DB 제약).

CRITICAL 해제 시 QC Lead PIN + 공장장 승인 필수.

Hold > 7 days without Disposition → daily reminder; > 30 days → EXPIRED + Plant Manager escalation.
7일 미처분 일일 알림, 30일 시 EXPIRED + 공장장 에스컬레이션.
Escalate
BR-QC-17
Hold label barcode must be intact and re-scannable. Damage/missing scan triggers immediate QC alert.
Hold 라벨 바코드 무결 + 재스캔 가능 필수. 훼손/실패 시 QC 알림.
Alert

Hold/Release events with PIN/Reason/Approver retained ≥ 5 years (immutable audit log).
Hold/Release 이벤트 + PIN/사유/승인자 5년 이상 봉인 보관.
Retain
