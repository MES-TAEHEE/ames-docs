01Overview개요
EN · Description The core POP screen of the IMG module. After each wrapping cycle, operator enters good-quantity produced via large keypad. [CT] = standard cycle qty from MD_Recipe. On Confirm: (1) INSERT IMG_Production, (2) WO CompletedQty += qty, (3) fabric length auto-deducted = BOM unit × qty from current roll, (4) OEE Performance fed, (5) QC trigger at interval. Defects → IMG-05. Critical: must have a verified fabric roll mounted (BR-IMG-01) — otherwise screen is locked.
KO · 설명 IMG 모듈의 핵심 POP 화면. 감싸기 사이클 종료 후 양품 수량을 대형 키패드로 입력. [CT] = MD_Recipe 표준 사이클 수량. 확정 시: (1) IMG_Production 적재, (2) WO CompletedQty += qty, (3) 원단 잔량 자동 차감 = BOM 단위 × qty, (4) OEE 성능률 데이터, (5) 검사 주기 도달 시 QC 트리거. 불량 발생 시 IMG-05. 핵심: 검증된 원단 롤 장착 필수(BR-IMG-01) — 미장착 시 화면 잠금.

03Auto Data Flow on Confirm확정 시 자동 연계
TargetUpdate
IMG_ProductionINSERT (EntryID, WOID, GoodQty, FabricRollID, FabricConsumedM, CycleSec, BondTempAvg, EmpID).
PP_WOCompletedQty += GoodQty. ≥ OpenQty 시 Status → Closed.
WH_Stock (Fabric)RemainingM -= FabricConsumedM. ≤ 10m 시 노랑 알림, ≤ 5m 시 IMG-04 재투입 권고.
OEEPerformance Rate 분자에 GoodQty 적재 + 이론 CT 대비 실제 CycleSec 비율.
QCWO 누적 양품이 MD_InspStd.SampleInterval 도달 시 QC-01 in-process 자동 발행.
Hourly Stats CacheIMG-02 Dashboard용 5초 캐시 갱신.
04Workflow작업 흐름
Cycle completes (PLC reports cycle end signal). Bond temp recorded.
사이클 종료 (PLC 신호). 본드 온도 자동 기록.
Operator inspects parts at-line. All good → [CT] tap (auto-fill) → Confirm.
작업자 라인내 점검. 양품 시 [CT] 자동 채움 → Confirm.
Defect found → manual qty → [Defect] → IMG-05.
불량 발생 시 양품 수동 입력 → [Defect] → IMG-05.
Confirm: 6 auto-updates fire within 200ms. ✓ flash + chirp.
Confirm: 6개 자동 갱신 200ms 내. ✓ + chirp.
Fabric remaining auto-recalculated; if < 5m → IMG-04 prompt next cycle.
원단 잔량 자동 재계산. 5m 미만 시 다음 사이클부터 IMG-04 권고.
