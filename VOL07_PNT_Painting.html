도장 생산 POP · 5-Stage Forced-Sequence Process Control
POP system for 5-stage interior painting process. Stages must be completed in strict order: Pre-Treatment → Primer → Top Coat → Oven Cure → Inspection. Oven temperature is monitored continuously (80°C ±5°C, 90 min). Temperature deviation triggers Andon + auto MNT work-order + QC enhanced inspection flag. 7 screens.

SEQUENCE ENFORCEMENT Each stage is locked by default. After an operator completes and confirms a stage, the next stage unlocks. Skipping or reversing order is technically impossible. If any stage fails QC criteria, the WO is flagged for enhanced inspection (QC module).

OVEN TEMPERATURE MONITOR
Real-time oven temperature chart — deviation ±5°C triggers Andon + MNT work order
건조로 온도 실시간 차트 — ±5°C 이탈 시 Andon + MNT 작업지시 자동 생성
ALARM CHAIN If oven temperature deviates beyond 80°C ±5°C for more than 2 consecutive minutes: (1) Andon alarm sounds, (2) MNT maintenance work order created automatically (MNT module), (3) the affected WO is flagged for QC enhanced inspection (QC-02 cannot be bypassed). All deviation events logged with timestamp for traceability.
경보 체인 온도 80°C ±5°C 2분 이상 이탈 시: (1) Andon 경보 발령, (2) MNT 작업지시 자동 생성, (3) 해당 WO QC 강화검사 플래그(QC-02 우회 불가). 모든 이탈 이벤트 타임스탬프 포함 이력 기록.
