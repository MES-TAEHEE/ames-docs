5-STAGE
PAINTING POP
도장 생산 POP
POP and Web screens for the 5-stage painting process applied to bumpers: Pre-treatment → Base Coat → Clear Coat → Oven Drying → Final Inspection. Includes color recipe setup from MD-10, real-time oven temperature monitoring with IoT sensors (target 80±5°C for 90 min), immediate Andon on temperature deviation, automatic QC enhanced-inspection flag on deviation WOs, and 6 PNT-specific defect types.
범퍼 5단계 도장 POP+Web: 전처리→베이스→클리어→건조로→검사. MD-10 색상 레시피 세팅, IoT 건조로 실시간 온도 모니터(목표 80±5°C/90분), 이탈 시 Andon 즉시 발동, 이탈 WO QC 강화검사 플래그 자동 설정, PNT 전용 불량 6종.

PNT-03
5-Stage Production Entry
5단계 도장 실적 입력
🏭 POP Dark2nd Dev
EN
Production is entered stage-by-stage in mandatory sequence: Pre-treatment → Base Coat → Clear Coat → Oven Drying → Final Inspection. Skipping a stage is blocked. Each stage records start time, completion time, operator, and quantity. Oven stage automatically monitors temperature via IoT sensor integration.

PNT-07
Oven Temperature Monitor
건조로 실시간 온도 모니터링
🖥 Web PC2nd Dev
EN
Monitors real-time oven temperature from IoT sensors with 10-second polling. Target: 80°C ± 5°C for 90 minutes per WO batch. When temperature deviates beyond ±5°C from target, three automatic actions are simultaneously triggered: (1) Andon alarm on PNT POP, (2) MNT deviation log auto-created, (3) QC enhanced-inspection flag set on all WOs in that oven batch.
KO
IoT 센서 10초 주기 건조로 실시간 온도 모니터링. 목표 80±5°C / 90분. ±5°C 이탈 시 3가지 자동 동시 실행: ① PNT POP Andon 알람 ② MNT 이탈 이력 자동 생성 ③ 해당 배치 전체 WO QC 강화검사 플래그 자동 설정.
