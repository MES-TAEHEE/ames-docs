01Overview개요
EN · Description The core quality gate of A-MES — the last barrier between production and finished goods. WO completion at the last process triggers QC-03 automatically. Inspection checklist loaded from MD-08 (item-specific). Standard sampling OR Enhanced 100% inspection if upstream flagged (PNT oven deviation, INJ mold-wear flag, prior shift Critical FAIL). PASS → FG-01 stocking auto-triggered. FAIL → output quarantined to QC-05 Hold + NCR(QC-04) + cannot ship. Customer audit-grade record retention.
KO · 설명 A-MES의 핵심 품질 게이트 — 생산과 완제품 사이의 마지막 방벽. 마지막 공정 WO 완료 시 QC-03 자동 발행. 품목별 MD-08 체크리스트 로드. 표준 샘플링 또는 강화 100% 전수검사(PNT 오븐 이탈, INJ 금형 마모, 이전 교대 Critical FAIL 등 상류 플래그 시). 합격 → FG-01 적재 자동 트리거. 불합격 → QC-05 격리 + NCR(QC-04) + 출하 차단. 고객 감사 대응 등급 보관.

03FG Gate CascadeFG 게이트 자동 연계
EventAction
Auto TriggerWO의 마지막 공정 완료(예: PNT-07 LABELED) 시 QC-03 자동 발행. 상류 플래그 검사 → Mode = STANDARD / ENHANCED 자동 결정.
Enhanced Trigger Sources(1) PNT-05 Oven Deviation Flag, (2) INJ Mold ≥ 80% + FLASH defect, (3) PNT-08/INJ-05 Critical defect 발생 LOT, (4) 이전 24h 내 동일 WO QC-02 FAIL, (5) 고객 별도 요청.
PASS(1) FG-01 PDA 적재 트리거 자동 발사 + FGTriggerID 발급, (2) PP_WO Status: Inspect → Closed, (3) 고객 출하 가능 상태 ON, (4) RPT FPY(First Pass Yield) 데이터 적재.
FAIL(1) QC-05 Hold 자동 생성 (전체 BatchQty), (2) NCR(QC-04) 자동 발행 + 사진/측정값 자동 첨부, (3) FG-01 차단 (BR-QC-01), (4) 영업/생산팀 알림, (5) Customer notify 옵션 활성.
MARGINALQC Lead 24h 내 재검 필수. 미결정 시 자동 FAIL 전환 (BR-QC-04). 재검에서 PASS 시 정상 경로, FAIL 시 위 FAIL 경로.
3회 FAIL (30일)동일 WO 또는 동일 품목 + 동일 불량코드 3회 → CAPA(QC-06) 자동 권고 (BR-QC-03).
04Workflow작업 흐름
Final process completes for WO (PNT LABELED or last process Closed) → QC-03 auto-issued.
WO의 마지막 공정 완료(PNT LABELED 등) → QC-03 자동 발행.
System evaluates upstream flags and decides Mode = STANDARD (AQL sample) or ENHANCED (100%).
상류 플래그 평가 후 Mode = STANDARD(AQL) 또는 ENHANCED(전수) 자동 결정.
QC Final Inspector opens job on Web; checklist auto-loaded from MD-08 (customer-spec compliant).
QC 최종 검사원이 Web에서 검사 시작, MD-08 체크리스트 자동 로드 (고객사양 준수).
Inspector enters per-item measurements + photos. System auto-judges each item; one FAIL → batch FAIL.
항목별 측정값 + 사진 입력. 시스템이 항목별 합/부 자동 판정, 1건이라도 FAIL이면 배치 FAIL.
MARGINAL items routed to QC Lead for final disposition within 24h.
MARGINAL 항목은 QC Lead에게 routing, 24h 내 최종 판정.
PASS: FG-01 stocking auto-fires within 1s. FAIL: QC-05 Hold + NCR auto-cascades; FG-01 blocked at DB.
PASS 시 FG-01 적재 1초 내 자동 발사. FAIL 시 QC-05 Hold + NCR 자동, FG-01은 DB level 차단.
All measurements, photos, decisions, and approver IDs are sealed for ≥ 5 year audit trail.
모든 측정·사진·결정·승인자 ID는 5년 이상 감사 기록으로 보관.
05Business Rules업무 규칙
RuleDescriptionAction
BR-QC-01
Only QC-03 PASS WOs can proceed to FG-01 stocking. Constraint enforced at DB level.
QC-03 합격 WO만 FG-01 적재 가능 (DB 제약).
Block
BR-QC-02
WOs auto-flagged for ENHANCED inspection cannot be manually downgraded to STANDARD.
강화검사 자동 지정 시 표준 샘플링으로 수동 변경 불가.
Forced
BR-QC-04
MARGINAL must be resolved by QC Lead within 24h; otherwise auto-converts to FAIL.
MARGINAL 24h 내 QC Lead 미판정 시 FAIL 자동 전환.
Escalate
BR-QC-03
3 FAILs of same defect within 30 days → CAPA recommendation auto-generated.
30일 내 동일 불량 FAIL 3회 → CAPA 자동 권고.
Alert
BR-QC-10
Final inspection requires Lead-or-above approver for PASS verdict on Customer-A grade items.
고객 A등급 품목은 PASS 판정 시 QC Lead 이상 승인자 필수.
Approve
BR-QC-08
Full inspection record (measurements, photos, signatures) retained ≥ 5 years.
전 측정값·사진·서명 5년 이상 보관 (IATF/감사).
Retain
