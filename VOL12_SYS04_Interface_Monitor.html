01Overview개요
EN · Description The Interface Monitor shows the live connection status of every external interface A-MES depends on — primarily the SCM order feed and ERP cost / master sync. Each interface is shown with a status light (green = OK, amber = degraded, red = error), last successful sync time, record count, and recent error log. When an interface fails or its last sync exceeds the allowed gap, the screen flags it red and a notification fires (SYS-06). It lets admins detect and react to integration breakdowns before they disrupt production. Auto-refresh every 60 seconds.
KO · 설명 인터페이스 모니터는 A-MES가 의존하는 모든 외부 인터페이스의 실시간 연결 상태를 보여준다 — 주로 SCM 주문 수신과 ERP 원가/마스터 동기. 각 인터페이스는 상태 표시등(초록=정상, 주황=저하, 적색=오류), 마지막 성공 동기 시각, 레코드 수, 최근 오류 로그와 함께 표시된다. 인터페이스가 실패하거나 마지막 동기가 허용 간격을 초과하면 화면이 적색 표시하고 알림(SYS-06)이 발생한다. 통합 장애가 생산을 방해하기 전에 감지·대응하게 한다. 60초 자동 갱신.
📌 Early Warning · 조기 경보SCM 주문 피드가 끊기면 PP가 신규 주문을 받지 못한다 — SYS-04는 이를 즉시 적색으로 드러내어 수동 재동기·복구를 유도한다. · If the SCM feed stops, PP receives no new orders — SYS-04 surfaces it instantly for manual re-sync.

04Workflow작업 흐름
Each interface syncs on its schedule; the monitor records the result and timestamp.
각 인터페이스가 스케줄대로 동기 — 모니터가 결과·시각을 기록한다.
The screen auto-refreshes every 60 seconds, showing live status lights.
화면이 60초마다 자동 갱신되어 실시간 상태등을 표시한다.
On a sync failure the system auto-retries; if last sync exceeds MaxGap, status → ERROR.
동기 실패 시 자동 재시도 — 마지막 동기가 허용 간격 초과 시 ERROR.
An ERROR fires a notification (SYS-06) to Admin; the error is logged.
ERROR 발생 시 Admin에게 알림(SYS-06) 발생, 오류 로그 기록.
Admin investigates, triggers a Manual Sync; on success the status returns to OK.
Admin이 조사 후 수동 재동기 실행 — 성공 시 상태가 OK로 복귀한다.
05Business Rules업무 규칙
RuleDescriptionAction
BR-SYS-16
If an interface's last sync exceeds MaxGapMinutes, the status auto-changes to ERROR.
마지막 동기가 허용 간격 초과 시 상태 자동 ERROR.
Auto
BR-SYS-17
A failed sync is auto-retried a fixed number of times before ERROR is confirmed.
실패한 동기는 정해진 횟수 자동 재시도 후 ERROR 확정.
Retry
BR-SYS-18
An interface ERROR fires a notification to Admin and is written to the audit log.
인터페이스 ERROR 시 Admin 알림 + 감사 로그 기록.
Notify
BR-SYS-19
Manual Sync is Admin-only; every manual sync is logged with actor and result.
수동 재동기는 Admin 전용 — 주체·결과 로그 기록.
Restrict
