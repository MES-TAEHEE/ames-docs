01Overview개요
EN · Description The Daily Production Report consolidates the day's manufacturing output across all lines (INJ, PNT) into a single read-only summary. For each line / item / Work Order it shows planned qty, actual good qty, defect qty, defect rate, and yield, with a shift breakdown (Day / Night). It is auto-generated when FG-07 Day-End Close is executed — no manual trigger. The report is a frozen snapshot; figures cannot be edited unless the day is re-opened by a supervisor. Each row drills down to the source Work Order and shift records. Excel / PDF export and optional email dispatch are supported.
KO · 설명 일별 생산 실적 보고서는 전 라인(INJ·PNT)의 당일 생산을 단일 읽기전용 요약으로 통합한다. 라인·품목·작업지시(WO)별로 계획 수량, 실적 양품 수량, 불량 수량, 불량율, 수율을 표시하고 교대(주간/야간) 분해를 포함한다. FG-07 일 마감 실행 시 자동 생성되며 수동 트리거가 없다. 보고서는 고정 스냅샷으로, 관리자가 일 마감을 재오픈하지 않는 한 수치를 수정할 수 없다. 각 행은 원천 WO·교대 기록으로 드릴다운된다. Excel/PDF 출력과 이메일 발송을 지원한다.
📌 Report Position · 보고서 위치RPT-01 is one of two daily-auto reports (with RPT-03 Shipment). It feeds the RPT-07 Monthly KPI Dashboard's "Plan Achievement" metric and is the primary daily production review document for the morning meeting. · RPT-01은 일별 자동 보고서 2종 중 하나(RPT-03 출하와 함께)로, RPT-07 월간 KPI의 '계획 달성율' 지표에 집계되며 조회의 핵심 생산 검토 문서이다.

04Workflow작업 흐름
Throughout the day, INJ/PNT WO results and QC judgments accumulate in transaction tables.
하루 동안 INJ/PNT WO 실적과 QC 판정이 트랜잭션 테이블에 누적된다.
Before close, users may preview the report — shown with a PRELIMINARY watermark.
마감 전 사용자는 보고서를 미리 볼 수 있다 — PRELIMINARY(잠정) 워터마크 표시.
FG-07 Day-End Close is executed → RPT-01 aggregation job runs automatically.
FG-07 일 마감 실행 → RPT-01 집계 작업이 자동 수행된다.
Job aggregates by line/item/WO/shift, computes defect rate & yield, writes a frozen snapshot (Status FINAL).
라인·품목·WO·교대별 집계, 불량율·수율 산출, 고정 스냅샷 기록 (Status FINAL).
Report becomes available in RPT-09 Report Center; configured recipients receive auto-email.
보고서는 RPT-09 리포트 센터에 등록되고, 설정된 수신자에게 자동 이메일 발송된다.
If the day is re-opened by a supervisor, the report reverts to PRELIMINARY and is re-generated on the next close.
관리자가 일 마감을 재오픈하면 보고서는 PRELIMINARY로 되돌아가고 다음 마감 시 재생성된다.
05Business Rules업무 규칙
RuleDescriptionAction
BR-RPT-04
RPT-01 is auto-generated only by FG-07 Day-End Close — no manual generate button.
RPT-01은 FG-07 일 마감으로만 자동 생성 — 수동 생성 버튼 없음.
Auto
BR-RPT-05
A FINAL report is read-only; reported figures cannot be edited. Source correction requires day re-open (supervisor).
FINAL 보고서는 읽기전용 — 수치 수정 불가, 원천 정정은 일 재오픈(관리자) 필요.
Lock
BR-RPT-06
Item rows with defect rate ≥ 3% are highlighted; total defect rate ≥ 3% raises a flag on RPT-07.
불량율 ≥ 3% 품목 행 강조, 전체 ≥ 3%는 RPT-07에 플래그.
Flag
BR-RPT-07
Plan, good, and defect totals must reconcile with WO results; a mismatch blocks FINAL and alerts the closing user.
계획·양품·불량 합계는 WO 실적과 정합 — 불일치 시 FINAL 차단·마감자 경고.
Block
