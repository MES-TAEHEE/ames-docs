01Overview개요
EN · Description Category D supplies the masters for the RFID-tracked powder painting line. RFID Tag (MD-16) registers the UHF Gen2 tags permanently bonded to jigs — heat-rated to survive the 230°C oven. RFID Reader (MD-19) registers the 3 fixed gate readers (R1 Load · R2 Oven · R3 Unload) per line. Paint / Fabric (MD-10) covers liquid paint, powder, fabric and adhesive material LOTs with expiry. RAL Paint Color (MD-17) holds the RAL color codes, current powder LOT, cure spec and electrostatic recipe parameters that drive the PNT-01 color recipe.
KO · 설명 카테고리 D는 RFID 추적 분체도장 라인의 마스터를 공급한다. RFID 태그(MD-16)는 지그에 영구 부착된 UHF Gen2 태그를 등록한다 — 230°C 오븐을 견디는 내열 사양. RFID 리더(MD-19)는 라인당 3개 고정 게이트 리더(R1 로딩 · R2 오븐 · R3 언로딩)를 등록한다. 도료/원단(MD-10)은 액상 도료·분체·원단·접착제 LOT을 유효기간과 함께 다룬다. RAL 컬러(MD-17)는 RAL 코드, 현재 분체 LOT, 경화 스펙, 정전 레시피 파라미터를 보유하며 PNT-01 컬러 레시피를 구동한다.
📌 RFID Survives the Oven · 오븐을 견디는 RFIDMD-16 태그의 HeatRating은 오븐 목표온도 + 30°C 마진 이상이어야 한다 — 230°C 경화 오븐을 통과하면서도 EPC를 유지해야 분체도장 추적이 끊기지 않는다. · A tag's HeatRating must exceed the oven temp + 30°C margin so RFID tracking survives the 230°C cure.

04Maintenance Workflow마스터 유지보수 흐름
RFID tags are heat-rated, registered, and bonded to jigs; the EPC is recorded against the Jig.
RFID 태그는 내열 검증 후 등록·지그에 부착, EPC를 지그에 기록한다.
3 readers per RFID line are installed and registered (R1/R2/R3) before PNT line go-live.
RFID 라인당 리더 3개(R1/R2/R3)를 설치·등록한 뒤 PNT 라인을 가동한다.
Paint / Fabric material is registered per LOT with expiry; expired LOTs are auto-blocked.
도료/원단 자재를 LOT별 유효기간과 함께 등록 — 만료 LOT은 자동 차단.
RAL colors are defined with cure spec and recipe; PowderLot is mapped to a valid MD-10 material.
RAL 컬러를 경화 스펙·레시피와 함께 정의, PowderLot을 유효 MD-10 자재에 매핑한다.
Tag CycleCount auto-increments at each oven pass; tags near end-of-life are flagged for replacement.
태그 CycleCount는 오븐 통과 시마다 자동 증가 — 수명 임박 태그는 교체 플래그.
05Business Rules업무 규칙 (해당 카테고리)

BR-MD-09
MD-16 Tag HeatRating must be ≥ MD-18 Oven TargetTemp + 30°C safety margin.
태그 내열등급 ≥ 오븐 목표온도 + 30°C 마진.

Each RFID line must have exactly 3 readers (R1/R2/R3) registered before PNT line go-live.
RFID 라인은 리더 3개 등록 후 가동.

BR-MD-07
MD-17 RAL color PowderLot must be mapped to an MD-10 paint material with a valid expiry.
RAL 컬러 PowderLot은 유효 MD-10 도료 LOT과 매핑.
