01Overview개요
EN · Description SYS-06 configures how A-MES alerts people to important events. For each of 8 event types (low stock, Andon alert, mold-life warning, oven deviation, overdue order, account lock, PM due, CAPA recommend) the Admin sets whether it is enabled, the delivery channel(s) — In-App / Email / SMS — and the recipient roles. Each event toggles independently, and changes take effect immediately. The screen is the single control point for notification policy; the actual sending is handled by the SYS notification engine and email/SMS queue.
KO · 설명 SYS-06은 A-MES가 중요 이벤트를 사람에게 어떻게 알릴지 구성한다. 8종 이벤트(재고 부족·Andon 알람·금형 수명 경고·오븐 이탈·납기 초과·계정 잠금·PM 예정·CAPA 권고) 각각에 대해 Admin이 활성화 여부, 전달 채널(인앱/이메일/SMS), 수신 역할을 설정한다. 이벤트별로 독립 토글되며 변경은 즉시 반영된다. 화면은 알림 정책의 단일 통제점이며, 실제 발송은 SYS 알림 엔진과 이메일/SMS 큐가 처리한다.
📌 Channel Escalation · 채널 강도이벤트 긴급도에 따라 채널을 조합한다 — 일반은 인앱·이메일, 긴급(Andon·오븐 이탈)은 SMS까지 추가해 즉시 도달을 보장한다. · Channels are layered by urgency — SMS is added for critical events (Andon, oven deviation) to guarantee immediate reach.

04Workflow작업 흐름
Admin opens SYS-06; the 8 event types are listed with current channel and recipient settings.
Admin이 SYS-06을 연다 — 8종 이벤트가 현재 채널·수신자 설정과 함께 나열된다.
Admin toggles an event on/off, edits its channels and recipient roles.
Admin이 이벤트를 켜고/끄고, 채널·수신 역할을 편집한다.
On Save, the new policy takes effect immediately; the change is logged in SYS-05.
저장 시 새 정책이 즉시 반영 — 변경은 SYS-05에 기록된다.
When a source module raises an event, the notification engine checks SYS-06 policy.
원천 모듈이 이벤트를 발생시키면 알림 엔진이 SYS-06 정책을 확인한다.
If enabled, messages are sent on the configured channels to the recipient roles.
활성 상태면 설정된 채널로 수신 역할에게 메시지가 발송된다.
05Business Rules업무 규칙
RuleDescriptionAction
BR-SYS-24
Each of the 8 event types is toggled independently; a disabled event sends nothing.
8종 이벤트는 각각 독립 토글 — 비활성 이벤트는 발송 없음.
Config
BR-SYS-25
Notification policy changes take effect immediately on save — no restart required.
알림 정책 변경은 저장 즉시 반영 — 재기동 불필요.
Immediate
BR-SYS-26
Recipients are specified by role (SYS-02); all users holding that role receive the alert.
수신자는 역할(SYS-02) 단위 — 해당 역할 보유 사용자 전원 수신.
Auto
BR-SYS-27
Critical events (Andon, oven deviation) cannot be fully disabled — at minimum In-App stays on.
긴급 이벤트(Andon·오븐)는 완전 비활성 불가 — 최소 인앱 유지.
Mandatory
