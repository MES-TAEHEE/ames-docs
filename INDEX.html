DESIGN SPEC
자동차 부품 제조 실행 시스템 · UI 설계서
Complete UI design specification for the A-MES (Automotive Manufacturing Execution System) for an automotive interior parts manufacturer supplying Seoyon E-Hua SAV (Savannah) and GEO (Georgia) plants. Covers 11 functional modules, 80+ screens, Web + PDA + POP Dark interfaces. All UI labels in English, descriptions in bilingual EN/KO.
서연이화 SAV(사바나)·GEO(조지아) 납품 자동차 내외장 부품 제조사를 위한 A-MES 전체 UI 설계서. 11개 기능 모듈, 80개 이상 화면, Web + PDA + POP 다크 인터페이스. UI 레이블 영문, 화면 설명 영한 병행.

Key Business Rules Summary · 핵심 업무 규칙 요약
BR-WH-02
All inbound triggers QC-01 incoming inspection automatically.
입고 즉시 수입검사 자동 트리거
BR-PP-02
WO cannot be released if MRP shows material shortage.
자재 부족 시 WO 릴리즈 차단
BR-INJ-03
Defect rate > 5% → Andon alarm auto-triggered.
불량율 5% 초과 → Andon 자동 발동
BR-IMG-02
Color mismatch on fabric scan → immediate block + alarm.
원단 색상 불일치 즉시 차단 + 경고음
BR-PNT-03
Oven deviation → QC enhanced inspection auto-flag on all WOs in batch.
건조로 이탈 → 해당 배치 WO 전수 강화검사 자동 플래그
BR-QC-01
Only QC-03 PASS WOs can proceed to FG stocking.
QC-03 합격 WO만 FG 적재 가능
BR-FG-02
FG outbound picking enforces FIFO by stocking date.
출고 피킹 입고일 기준 FIFO 강제
BR-MNT-02
Mold 90% shot life → yellow warning. 100% → forced change request.
금형 90% 경고 → 100% 교체 요청 자동
BR-RPT-01
All daily reports auto-generated on day-end close.
일 마감 시 일별 보고서 자동 생성
BR-SYS-01
5 consecutive login failures → account auto-locked.
로그인 5회 실패 → 계정 자동 잠금
BR-MD-01
No WO without BOM + BOP routing registered in Master Data.
BOM·BOP 미등록 품목 WO 생성 차단
BR-WH-04
All WH outbound follows FIFO. System auto-suggests oldest location.
WH 출고 FIFO 자동 적용, 최고령 위치 제안
