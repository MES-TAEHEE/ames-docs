01Overview개요
EN · Description MD_Line registers the physical production lines of the plant — injection, wrapping and the RFID powder-painting lines. Each line carries a type, a default Work Center, a daily capacity and a shift pattern, and an RfidEnabledFlag that marks the lines requiring the 3-reader RFID gate set. The Line is the unit at which PP plans, and at which the painting line's RFID tracking is scoped. Work Centers and Equipment roll up to a Line.
KO · 설명 MD_Line은 공장의 물리적 생산 라인을 등록한다 — 사출·감싸기·RFID 분체도장 라인. 각 라인은 유형, 기본 작업장, 일일 능력, 교대 패턴을 보유하며 3-리더 RFID 게이트 세트가 필요한 라인을 표시하는 RfidEnabledFlag를 가진다. 라인은 PP가 계획하는 단위이자 도장 라인의 RFID 추적 범위 단위다. 작업장·설비가 라인으로 집계된다.

🔮 Future Extension · 라인별 스케줄링 확장 예정현재 MD_Line은 라인 1개당 단일 ShiftPattern을 가진다. 향후 라인별 분단위 시간패턴(라인마다 다른 스케줄)과 가동·계획비가동·실제비가동 모니터링 연계를 위해 별도 테이블(예: MD_LineTimePattern)이 추가될 예정 — MD L3 완료 후 착수. · A separate MD_LineTimePattern table for per-line minute-level schedules + operating/downtime monitoring is planned, after MD L3.

05Validation & Business Rules검증 + 업무 규칙
RuleDescriptionAction
BR-MD-10
An RFID-enabled line must have exactly 3 readers (R1/R2/R3) registered before go-live.
RFID 라인은 리더 3개(R1/R2/R3) 등록 후 가동.
Block
VR-Line-01
RfidEnabledFlag may be TRUE only for a PAINTING line (CK_Line_Rfid).
RfidEnabledFlag는 PAINTING 라인만 TRUE 가능.
Validation
VR-Line-02
The DefaultWCID Work Center's LineID must point back to this line (consistency).
기본 작업장의 LineID는 본 라인을 역참조 (정합성).
Validation
BR-MD-23
A line in use cannot be deleted — set Status = RETIRED instead.
사용 중 라인은 삭제 불가 — Status=RETIRED 전환만.
Block
BR-MD-24
Line create / update requires an approval workflow + SYS-05 audit log.
라인 생성·수정은 승인 워크플로 + SYS-05 감사 로그.
Audit
