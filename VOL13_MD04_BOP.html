01Overviewк°ңмҡ”
EN В· Description MD_BOP (Bill of Process) defines the ordered sequence of operations an item passes through. Each row is one process step вҖ” a step sequence, a process code, the Work Center that performs it, and standard cycle / setup times. Steps are grouped by RoutingType вҖ” A (Wrapping: INJвҶ’IMGвҶ’QCвҶ’FG) or B (Powder Paint: INJвҶ’PNTвҶ’QCвҶ’FG). PP-03 copies the full BOP step set into a Work Order's routing on creation; a WO cannot be created without a BOP, and a planner cannot manually set a non-BOP routing.
