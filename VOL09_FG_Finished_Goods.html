FINISHED
GOODS
완제품 관리 — Stock · FIFO Pick · SAV/GEO 출하 · OTD 추적
End-to-end finished goods management. QC-03 PASS-driven stocking (FG-01) with Slotting algorithm. 5-Status inventory (Available/Reserved/Hold/Shipped/Returned) with live WebSocket. EDI/Portal order intake (FG-03) with FIFO atomic Reservation. Mandatory FIFO picking (FG-04) with DB-level enforcement. Chain-of-Custody loading (FG-05) with driver e-signature + GPS. Customer-format delivery notes (FG-06 SAV/GEO). Day-End immutable snapshot (FG-07) feeding ERP. 5y retention audit history (FG-08). Real-time KPI dashboard (FG-09). RMA-driven return processing (FG-RTN) with 4-way disposition.
완제품 전 흐름 관리. QC-03 PASS 기반 적재(FG-01) Slotting 알고리즘. 5-Status 재고(가용/예약/격리/출하/반품) 라이브 WebSocket. EDI/포털 주문(FG-03) FIFO 원자 예약. FIFO 강제 피킹(FG-04) DB-level. Chain-of-Custody 상차(FG-05) 전자서명 + GPS. 고객 양식 거래명세서(FG-06 SAV/GEO). 일 마감 봉인 스냅샷(FG-07) → ERP. 5년 봉인 이력(FG-08). 실시간 KPI 대시보드(FG-09). RMA 반품 처리(FG-RTN) 4-way 처분.
