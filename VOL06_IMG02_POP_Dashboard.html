01Overview개요
EN · Description Default screen after IMG-01 login. Real-time wrapping line dashboard with large fonts readable from 5 meters: WO progress, current fabric roll status (color + remaining length), bond machine status, defect rate gauge, hourly output bar chart. Unique to IMG: visual fabric roll indicator with color swatch matching WO requirement. Tap any tile to drill into IMG-03/04/05/06.
KO · 설명 IMG-01 로그인 후 기본 화면. 5m 거리 가독 대형 폰트 실시간 감싸기 라인 현황판: WO 진도, 현재 원단 롤 상태(색상 + 잔량), 본드기 상태, 불량률 게이지, 시간별 생산 차트. IMG 고유: WO 요구 색상과 매칭되는 원단 롤 시각 표시. 타일 터치로 IMG-03/04/05/06 진입.
