WRAPPING POP
감싸기 생산 POP — 7 Screens · Color Verify · Auto Fabric Deduction
POP screens for the IMG (Fabric Wrapping) process — bonding fabric or leather to injection-molded plastic substrates. Key differentiators: mandatory fabric roll QR scan before mounting, automatic color-code verification against WO requirements (wrong color = instant block + alarm), BOM-based automatic fabric length deduction per production cycle, 6 IMG-specific defect types, and bond machine recipe parameter management.
사출 기재에 원단/가죽을 접착하는 IMG 감싸기 공정 POP. 핵심: 원단 롤 QR 스캔 필수, WO 요구 색상 자동 검증(불일치 즉시 차단 + 경고음), BOM 기반 원단 잔량 자동 차감, IMG 전용 불량 6종, 본드 기계 레시피 파라미터 관리.

Entry point to every IMG POP terminal. Operators authenticate via employee badge barcode scan (primary, ~1.5s) or 4-digit PIN (backup). Line and shift auto-detected from MD calendar roster. Identical UX to INJ-01 with IMG violet theme. 3 failed attempts → 30s lockout + supervisor alert.

모든 IMG 감싸기 라인 POP 터미널의 진입점. 사번 배지 바코드 스캔(기본, ~1.5초) 또는 4자리 PIN(백업) 인증. MD 캘린더 로스터로 라인·교대 자동 설정. INJ-01과 동일 UX, IMG 보라 테마. 3회 실패 시 30초 잠금 + 라인장 알림.

IMG-03
Production Entry
감싸기 실적 입력 — Core
🏭 POP Dark⚙ Core
📋 L3 Detail →
EN · Description
Core POP screen of the IMG module. After each wrapping cycle, operator enters good qty via 72px numpad. [CT] = standard cycle qty from MD_Recipe. Confirm triggers 6 auto-updates: IMG_Production INSERT, WO progress, fabric length auto-deducted from current roll (BOM × qty), OEE feed, QC trigger, dashboard cache. Critical: verified fabric roll required (BR-IMG-01).
KO · 설명
IMG 모듈의 핵심 POP 화면. 감싸기 사이클 종료 후 양품 수량을 72px 숫자패드로 입력. [CT] = MD_Recipe 표준 사이클 수량. Confirm 시 6개 자동 갱신: IMG_Production 적재, WO 진도, 원단 잔량 자동 차감(BOM × qty), OEE 데이터, QC 트리거, 대시보드 캐시. 핵심: 검증된 원단 롤 필수(BR-IMG-01).
IMG-04
Fabric Input & Color Verify
원단 롤 투입·색상 검증 — Signature
📱 PDA⚙ Signature
📋 L3 Detail →
EN · Description
The signature differentiator of the IMG module. Operator scans fabric roll QR via PDA before mounting. System cross-checks color code against WO requirement: Match → proceed; Mismatch → block + red screen + alarm (BR-IMG-02). After confirmed input, fabric length auto-deducted per IMG-03 production confirm. PDA 4-step flow: Scan → Match/Mismatch → Confirm → Auto-Deduction. Eliminates wrong-color defects.
KO · 설명
IMG 모듈의 시그니처 차별화 화면. 작업자가 원단 롤 장착 전 PDA로 QR 스캔. 시스템이 색상 코드를 WO 요구와 즉시 대조: 일치 → 진행. 불일치 → 차단 + 빨간 화면 + 경고음(BR-IMG-02). 투입 확정 후 IMG-03 실적 확정마다 원단 잔량 자동 차감. PDA 4단계: 스캔 → 일치/불일치 → 확정 → 자동 차감. 오색 투입 불량 제로화.
IMG-05
Defect Register
불량 등록 — 6종 + 5% Andon
🏭 POP Dark1st Dev
📋 L3 Detail →
EN · Description
Entered from IMG-03 [Defect] button. Operator picks 1 of 6 IMG defect types (LIFTING / WRINKLE / BOND_BLEED / FAB_SOIL / COLOR_DIFF / CUT_DEFECT) via large tap-buttons + numpad qty. Auto rate calculation; ≥ 5% fires Andon. Special auto-links: COLOR_DIFF → FabricInputAttempt history; CUT_DEFECT × 3 → cutter replacement MNT WO.
KO · 설명
IMG-03 [Defect] 버튼에서 진입. 6종 IMG 불량(들뜸/주름/본드번짐/오염/색차/재단) 중 1개 + 숫자패드 수량. 불량률 자동 계산, 5% 시 Andon 자동 발동. 특수 자동 연계: COLOR_DIFF → 원단 투입 시도 이력 자동 표시, CUT_DEFECT 연속 3회 → 커터 교체 MNT WO 권고.
IMG-06
Bond Setup
본드 세팅 — Recipe + PLC
🏭 POP Dark2nd Dev
📋 L3 Detail →
EN · Description
Bond machine recipe parameter screen. Sets pressure (3.5–5.0 bar) / temperature (85–95°C) / hold time (90–150 sec) setpoints. Auto-loaded from MD_Recipe on WO Accept; operator can adjust within MD-defined tolerance. PLC realtime feedback. Out-of-tolerance changes require supervisor PIN + reason (audit log 7 years). Per-cycle measured values feed QC enhanced inspection.
KO · 설명
본드 기계 레시피 파라미터 화면. 압력(3.5–5.0 bar)·온도(85–95°C)·시간(90–150초) 설정값 관리. WO 수락 시 MD_Recipe 자동 로드, 작업자가 허용오차 내 조정 가능. PLC 실시간 피드백. 허용오차 외 변경 시 라인장 PIN + 사유 필수(감사 7년). 사이클별 실측값은 QC 강화검사에 활용.
