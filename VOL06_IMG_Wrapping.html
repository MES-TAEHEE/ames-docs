POP screens for the IMG (Image / Fabric Wrapping) process — bonding fabric or leather to injection-molded plastic substrates. Key differentiators: mandatory fabric roll QR scan before mounting, automatic color-code verification against WO requirements (wrong color = instant block), BOM-based automatic fabric length deduction per production cycle, and 6 IMG-specific defect types.

IMG-04
Fabric Roll Input & Color Verification
원단 롤 투입 & 색상 검증
🏭 POP Dark2nd Dev
EN · Description
Before mounting a fabric roll onto the wrapping machine, the operator scans the roll's QR code via PDA. The system instantly cross-checks the scanned fabric color code against the WO's required color. If the color matches, the operator can proceed with mounting and confirm the input. If the color does not match, input is immediately blocked — the screen turns red and an alarm sounds — preventing wrong-color defects. After each production cycle, the consumed fabric length is automatically deducted from warehouse inventory based on the BOM unit consumption rate.
KO · 설명
원단 롤을 감싸기 기계에 장착하기 전, 작업자가 롤의 QR코드를 PDA로 스캔. 시스템이 스캔된 색상코드를 WO 요구 색상과 즉시 대조. 색상 일치 시 투입 확정 진행. 색상 불일치 시 화면 빨간색 전환 + 경고음 즉시 차단 → 원단 오투입 불량 제로화. 생산 사이클마다 BOM 단위 소비량 기준 원단 잔량 WH 재고에서 자동 차감.
