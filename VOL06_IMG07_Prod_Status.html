01Overview개요
EN · Description Web-based supervisor dashboard summarizing all IMG lines: KPI tiles (Today Good, Defect, Avg OEE, Active WOs, Low-Stock Rolls), per-line table with fabric roll info + remaining length + bond temperature, hourly output chart. IMG-specific: visual color swatch column + fabric runway bar for each line. Drill-down to per-WO history, fabric consumption log. Excel export for shift handover.
KO · 설명 모든 IMG 라인을 요약하는 Web 라인장 대시보드: KPI 타일(금일 양품·불량·평균 OEE·진행 WO·잔량부족 롤), 라인별 테이블(원단 롤 정보 + 잔량 + 본드 온도), 시간별 생산 차트. IMG 고유: 컬러 스왓치 열 + 원단 잔량 막대. 드릴다운으로 WO 이력·원단 소비 로그. Excel 출력으로 교대 인수인계.

02KPI DefinitionsKPI 정의
KPIFormulaNotes
Today GoodSUM(IMG_Production.GoodQty) · 교대 시작 이후목표 = SUM(PP_WO.OpenQty / 교대 분배).
Today DefectSUM(IMG_Defect.Qty) · 같은 윈도우비율 = Defect / (Good + Defect) × 100.
Avg OEE(Σ A × P × Q) / N linesA=가동률, P=성능률, Q=품질률. 라인 평균.
Active WOsCOUNT(PP_WO WHERE Status='In Progress' AND Routing='A')Routing A IMG WO만.
Low Stock RollsCOUNT(WH_FabricStock WHERE RemainingM ≤ 10 AND Status='IN_USE')잔량 10m 이하 사용중 롤 개수.
Bond Temp Status실시간 PLC 평균 vs Setpoint ±Tolerance색상: 정상/임박/이탈.
