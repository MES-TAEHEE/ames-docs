01Overview개요
EN · Description MD_SparePart is the maintenance spare-parts catalog — a new master added to support the MNT module. Each part carries its category, a JSON list of compatible equipment, unit cost, safety stock, reorder point, lead time and supplier. When MNT-08 spare-parts stock drops to or below the ReorderPoint, an auto-reorder request is raised. Work Orders (MNT-07) consume parts against this catalog. Parts are deactivated, never deleted.
KO · 설명 MD_SparePart는 정비 자재 카탈로그로 — MNT 모듈을 지원하기 위해 추가된 신규 마스터다. 각 부품은 분류, 호환 설비 JSON 목록, 단가, 안전재고, 재발주점, 리드타임, 공급사를 보유한다. MNT-08 정비자재 재고가 ReorderPoint 이하로 떨어지면 자동 재발주 요청이 발생한다. 작업지시(MNT-07)가 이 카탈로그를 기준으로 자재를 소비한다. 부품은 비활성화하며 삭제하지 않는다.

05Validation & Business Rules검증 + 업무 규칙
RuleDescriptionAction
BR-MD-20
Spare-part stock at or below ReorderPoint triggers an MNT-08 auto-reorder of ReorderQty.
재고 ≤ 재발주점 시 MNT-08 자동 재발주 (ReorderQty 수량).
Auto
VR-Part-01
ReorderPoint must be ≥ SafetyStock (CK_Part_Reorder).
재발주점은 안전재고 이상.
Validation
VR-Part-02
CompatEquipJSON entries must reference valid EquipIDs in MD_Equipment.
호환 설비 JSON은 MD_Equipment 유효 설비만 참조.
FK
BR-MD-23
A part referenced by an open WO or stock record cannot be deleted — set ActiveFlag = FALSE.
진행 WO·재고 참조 부품은 삭제 불가 — 비활성화만.
Block

Spare-part create / update requires an approval workflow + SYS-05 audit log.
정비 자재 생성·수정은 승인 워크플로 + 감사 로그.
