VOL.3 · WH · Warehouse Management · 창고 관리
WAREHOUSE
MANAGEMENT
창고 관리
Manages all inbound receiving (PDA barcode scan), real-time inventory by location, stock adjustments, and outbound picking (FIFO). Interfaces with PP module for SCM-sourced PO schedules and with FG module for finished goods stocking.
PDA 바코드 스캔 기반 입고, 로케이션별 실시간 재고, 재고 조정, FIFO 출고 피킹 관리. PP 모듈 SCM 발주 연계, FG 완제품 적재 연동.

WH-01
Inbound Schedule
입고 예정
🖥 Web PC Phase 0 + 1st Dev
📋 L3 Detail →
EN · Description
Displays all open purchase orders received from the SCM portal and allows warehouse staff to register expected inbound schedules. ETA D-Day is auto-calculated; overdue deliveries are highlighted in red. Clicking "Receive" triggers the WH-02 PDA inbound flow.
KO · 설명
SCM 포탈 수신 발주 기반 입고 예정 등록 화면. 입고 ETA D-Day 자동 계산, 지연 건 빨간색 강조. [Receive] 클릭 시 WH-02 PDA 입고 흐름으로 연계.
WH-02
PDA Inbound Receiving
PDA 입고 처리
📱 PDA 1st Dev
📋 L3 Detail →
EN · Description
Warehouse staff scans item barcode via PDA, confirms quantity using the numeric keypad, then scans a shelf QR code to assign storage location. System validates barcode against PO. Mismatch is immediately blocked. Confirms stock entry and prints location label.
KO · 설명
창고 담당자가 PDA로 품목 바코드 스캔 → 숫자패드로 수량 입력 → 선반 QR 스캔으로 로케이션 지정. 바코드-PO 불일치 즉시 차단. 입고 확정 + 위치 라벨 출력.
WH-03
Inventory Status
재고 현황
🖥 Web PC 1st Dev
📋 L3 Detail →
EN · Description
Real-time inventory view by item, material type (Raw Material / Fabric / Paint), and location. Items below minimum stock level are highlighted with a red badge. Supports Excel export, and shows daily snapshot delta (vs. previous day).

품목·유형(원자재/원단/도료)·로케이션별 실시간 재고 현황. 최소재고 미달 항목 빨간색 강조. Excel 출력, 전일 대비 재고 증감 표시.
WH-04
Location Map
로케이션 맵
🖥 Web PC2nd Dev
📋 L3 Detail →
EN
Visual grid map of all warehouse zones and shelf locations. Each cell shows current item, quantity, and fill rate (color-coded: empty grey / partial yellow / full green / over-capacity red). Click cell to view location detail and stock history.
KO
창고 구역·선반 로케이션 시각적 그리드 맵. 각 셀에 현재 품목·수량·충전율(빈칸-회색/일부-노란/가득-초록/초과-빨강). 셀 클릭 시 로케이션 상세·재고 이력 조회.
ℹ️
Location code format: [Warehouse]-[Zone]-[Row]-[Col]-[Level] · e.g. WH-FAB-D-01-1 · 로케이션 코드 체계: [창고]-[구역]-[행]-[열]-[층]

Business Rules · 업무 규칙 (WH)
Rule ID Category 유형 Rule Description (EN) 규칙 설명 (KO) Action
BR-WH-01 Inbound Gate · 입고 통제 Items without a valid open PO cannot be received. Barcode scan is blocked immediately if no matching PO is found in the system. PO 없는 품목 입고 불가. 미등록 바코드 스캔 즉시 차단. Block
BR-WH-02 QC Trigger · 검사 트리거 All inbound items automatically trigger QC-01 incoming inspection. Inventory is updated only after QC pass. QC fail → items quarantined. 입고 완료 즉시 QC-01 수입검사 자동 트리거. 합격 후 재고 반영. 불합격 시 격리. Auto
BR-WH-03 Min Stock Alert · 최소재고 When on-hand quantity falls below the minimum stock level (defined in MD-09/10), an alert is sent to the warehouse manager and procurement team. 현재고 < 최소재고(MD-09/10 기준) 시 창고담당·구매팀 즉시 알림. Alert
BR-WH-04 FIFO · 선입선출 All outbound picking must follow FIFO (First In First Out) based on inbound date. The system auto-suggests the oldest stock location for each item. 출고 피킹 입고일 기준 FIFO 자동 적용. 최고령 재고 위치 자동 제안. Auto
