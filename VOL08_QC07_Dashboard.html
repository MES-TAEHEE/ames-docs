01Overview개요
EN · Description Real-time consolidated quality dashboard for QC Manager and Plant Manager. Top row of 6 KPI tiles (FPY, Defect Rate, Open NCRs, Active Holds, Open CAPAs, Avg Inspection Lead Time). Pareto chart of top defect codes (8-bar with % cumulative). 30/60/90-day trend lines for FPY and Defect Rate. Live list of Open NCRs and Active Holds with one-click drill-down. 4M Cause Pie. Suitable for both QC office wall TV (full-screen mode) and Web PC analytics view. Auto-refresh KPI 30s, charts 5min, live items (Hold/NCR) via WebSocket.
KO · 설명 QC 관리자·공장장용 실시간 통합 품질 대시보드. 상단 6개 KPI 타일 (FPY·불량률·미해결 NCR·활성 Hold·진행 CAPA·평균 검사 리드타임). 상위 불량 파레토 차트 (8-bar + 누적%). FPY/불량률 30/60/90일 추세선. 미해결 NCR + 활성 Hold 실시간 리스트 + 원-클릭 드릴다운. 4M 원인 파이차트. QC 사무실 벽면 TV (풀스크린)와 Web PC 분석 뷰 겸용. 자동 갱신: KPI 30초, 차트 5분, 실시간 항목(Hold/NCR) WebSocket.

02KPI SpecificationKPI 정의
KPIFormulaDescription & Target

03Refresh Strategy데이터 갱신 전략
ComponentRefreshMechanism
KPI Tiles30s증분 SQL 집계 + Redis 캐시. 카운트 변화 시 색상 부드러운 트랜지션.
Pareto / Cause Pie5minOLAP 큐브 미리 집계, 5분마다 재계산. 무거운 쿼리는 야간 배치.
Trend Lines1h일 집계 테이블 사용 (QC_DailyStats). 1시간마다 마지막 점만 갱신.
Active Holds / Open NCRsLive (WS)WebSocket: 새 Hold/NCR 생성 또는 상태 변경 시 즉시 push. 0.5초 내 화면 반영.
Wall TV Mode30s (rotation)3개 뷰 (KPI / Pareto / Live List) 30초씩 자동 회전. 풀스크린 + 큰 폰트.
04Workflow작업 흐름
QC Manager opens Dashboard at start of shift; checks 6 KPI tiles for color status.
QC Manager가 교대 시작 시 Dashboard 오픈, 6개 KPI 색상 확인.
Reviews Pareto for top defect; clicks bar → NCR list filtered for that defect code.
상위 불량 Pareto 검토, 막대 클릭 → 해당 불량 NCR 목록 필터.
Checks Active Holds for Aged items (≥7d red); drills into each via Hold ID link.
Active Holds 중 Aged 항목 (7일 이상 빨강) 확인, Hold ID 링크로 드릴다운.
Reviews Trend line direction over 90 days; if FPY drops or Defect rises, opens RPT for deeper analysis.
90일 추세 방향 검토, FPY 하락 또는 불량률 상승 시 RPT 모듈로 깊은 분석.
Sets Wall TV mode for QC office display; team members see live status without login.
Wall TV 모드 설정으로 QC 사무실 게시, 팀원이 로그인 없이 실시간 상태 확인.
Configures alert thresholds: e.g., FPY drops below 99% → Slack notification to QC Lead.
알림 임계 설정: 예) FPY 99% 미만 시 QC Lead Slack 알림.
End of shift: exports PDF report (KPI snapshot + Pareto + Active Holds list).
교대 종료 시 PDF 보고서 출력 (KPI 스냅샷 + Pareto + 활성 Holds 목록).
05Business Rules업무 규칙
RuleDescriptionAction
BR-QC-22
KPI tile color thresholds are managed centrally in MD-08 InspStd (per item/customer). Cannot be edited on the fly.
KPI 임계는 MD-08에서 중앙 관리, 즉석 편집 불가.
Lock
BR-QC-23
FPY drop > 1% week-over-week auto-alerts QC Lead + Plant Manager.
FPY 주간 1%p 이상 하락 시 QC Lead + 공장장 자동 알림.
Alert
BR-QC-24
Open NCR aged ≥ 7d highlights red in dashboard; ≥ 14d auto-escalates.
7일 이상 미해결 NCR 빨강 강조, 14일 이상 자동 에스컬레이션.
Escalate
BR-QC-25
Dashboard data must not include in-flight DRAFT records; only OPEN+.
DRAFT 상태 항목은 Dashboard 노출 제외, OPEN 이상만 표시.
Filter
BR-QC-26
Wall TV mode auto-rotates 3 views every 30s; cannot drill-in (read-only).
Wall TV 모드는 3개 뷰 30초 회전, 드릴다운 불가 (읽기 전용).
Lock
