PP-01
Customer Forecast Management
고객사 Forecast 수신 및 관리
OVERVIEWReceives and manages customer demand forecasts (M+1 to M+6) from SAV/GEO SCM Portal. Downloads Forecast Excel from the customer SCM portal and imports it into A-MES. Tracks gaps between forecast and confirmed orders.
개요고객사(SAV/GEO) SCM 포탈에서 Forecast Excel을 다운로드하여 A-MES에 Import한다. Forecast 대비 확정 수주 편차를 추적한다.

PP-02
SAP B1 Supply Plan Import
SAP B1 공급계획 수신
OVERVIEWReceives confirmed supply plans from SAP Business One. Current method: SAP B1 Excel Export → A-MES manual import. Future roadmap: SAP B1 Service Layer REST API auto-polling every 1 hour.
개요SAP Business One의 확정 공급계획(판매오더)을 A-MES로 수신한다. 현행: SAP B1 Excel Export → 수동 Import. 향후: SAP B1 Service Layer REST API 자동 폴링(1시간)으로 전환 예정.

PP-03
Production Planning & Confirmation
생산계획 수립 및 확인
OVERVIEWPLAN manager reviews the received supply plans against current FG inventory, equipment load, and Forecast reference. Confirms feasible items for WO batch creation.

PP-04
Work Order Management
작업지시 (WO) 관리
OVERVIEWManages Work Orders from Draft through Released, In Progress, to Closed. Displays routing type badge (A/B), progress bar, DDay indicator, and SAP B1 supply plan link.

PP-05
MRP — Material Requirements Planning
자재 소요 계획
OVERVIEWRuns MRP to calculate material shortage quantities. A-MES MRP handles real-time WO blocking. SAP B1 MRP handles mid/long-term procurement planning and PO generation.
개요A-MES MRP는 실시간 WO 차단·가용 여부 판단. SAP B1 MRP는 중장기 자재 소요·공급업체 발주 담당. 두 MRP를 역할 분리하여 병행 운영한다.

PP — Production Planning
PP-06
Purchase Requisition → SAP B1
자재발주 요청서 (PR) → SAP B1 전송
OVERVIEWGenerates Purchase Requisition records from MRP shortage results and transmits them to SAP B1 via Service Layer API (POST /PurchaseRequests). Tracks SAP B1 DocNum after successful transmission.
개요MRP 부족 자재를 기반으로 발주요청서(PR)를 생성하고, SAP B1 Service Layer API(POST /PurchaseRequests)로 전송한다. 전송 성공 시 SAP B1 DocNum을 수신하여 저장한다.

PP-07
Work Order Release
WO 발행 및 생산 착수
OVERVIEWValidates all release conditions before transitioning WO to Released status: Phase-0 complete, MRP no shortage, equipment available, delivery lead time sufficient.
개요WO Released 전환 전 4가지 조건을 순서대로 검증한다: Phase-0 완료, MRP 자재 충족, 설비 가용, 납기 여유. 모두 통과 시 WO 발행 → 자재 자동 예약 → POP 화면 노출.
