VOL.4 · PP · Production Planning · 생산계획
PRODUCTION
PLANNING
생산계획 — Forecast · SAP B1 Sync · MRP · Auto Purchase Req · WO Release
Full production planning cycle from customer demand intake to shop-floor release. Pipeline: customer Forecast (SCM portal) → SAP B1 supply plan import → MES production plan → MRP material requirements → Purchase Requisition (SAP B1 Service Layer API) → Work Order release → POP shop-floor (INJ/IMG/PNT). Dual MRP: A-MES for real-time WO blocking, SAP B1 for mid/long-term procurement.
고객 수요 수신부터 현장 발행까지 생산계획 전 사이클. 파이프라인: 고객 Forecast (SCM 포탈) → SAP B1 공급계획 Import → MES 생산계획 수립 → MRP 자재 소요 → 자재발주 요청서 (SAP B1 Service Layer API) → WO 발행 → POP 현장 (INJ/IMG/PNT). 이중 MRP: A-MES는 실시간 WO 차단, SAP B1은 중장기 발주 담당.
