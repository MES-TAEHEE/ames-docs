Concept  데모 사이트 개념
The A-MES demo site reorganizes the 11 design-spec modules into 3 demonstrable systems by usage environment, so each can be shown to its real audience on its real device. Office Web runs on desktop browsers for planners and engineers; Shop-Floor POP runs on fixed touch terminals beside the line; Logistics PDA runs on handheld scanners in the warehouse. This page is the structure design — the menu tree, screen inventory and navigation flow — produced before the interactive demo is built.
A-MES 데모사이트는 11개 설계 모듈을 사용 환경 기준 3개 시연 시스템으로 재편하여, 각 시스템을 실제 사용자·실제 디바이스로 시연할 수 있게 한다. Office Web은 계획자·엔지니어의 데스크톱 브라우저, Shop-Floor POP는 라인 옆 고정 터치 단말, Logistics PDA는 창고 핸드헬드 스캐너에서 구동된다. 본 페이지는 인터랙티브 데모 구축 전에 작성하는 구조 설계 — 메뉴 트리·화면 인벤토리·네비게이션 흐름 — 이다.
