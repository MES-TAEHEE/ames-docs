01Overview개요
EN · Description The entry point to every IMG POP terminal on the wrapping line. Operators authenticate via employee badge barcode scan (primary) or 4-digit PIN (backup). Line and shift are auto-detected from the roster (MD calendar + employee). After login, the POP transitions to IMG-02 Dashboard. Identical UX to INJ-01 but themed in IMG violet.
KO · 설명 모든 IMG 감싸기 라인 POP 터미널의 진입점. 사번 배지 바코드 스캔(기본) 또는 4자리 PIN(백업) 인증. 로스터(MD 캘린더 + 인원)로부터 라인·교대 자동 설정. 로그인 성공 시 IMG-02 현황판으로 전환. INJ-01과 동일 UX, IMG 보라 테마 적용.

02Authentication Methods인증 방법
MethodInputDescription · 설명
Badge Scan (Primary)Code-128사번 바코드(9자리). USB HID 스캐너 자동 입력. 가장 빠른 인증(~1.5초).
PIN (Backup)4-digit배지 분실·미발급 시 사번 4자리 입력. MD_Employee.PIN 비교.
Supervisor OverridePIN + Card관리자 PIN으로 일시적 작업자 변경(최대 30분, 감사 기록).
Failed Lockout—3회 연속 실패 → 30초 잠금 + 라인장 PDA 알림. 5회 누적 → 인사팀 통지.
03Session Management세션 관리
EventAction
Login SuccessSession 생성 → IMG-02 Dashboard 이동. Token 만료: 교대 종료 + 30분.
Idle Timeout30분 무입력 → 화면 잠금(로그아웃 아님). 본인 PIN 재입력 시 해제.
Shift Change교대 종료 시각 도달 시 강제 로그아웃. 진행 중 IMG-03 입력은 draft 보존.
Manual Logout명시적 로그아웃 시 상태 정리. IMG-04 미저장 원단 투입은 draft 유지.
Power/Network LossSQLite 로컬 캐시에 세션 유지. 복구 시 자동 재로그인 옵션(30분 이내).
04Workflow작업 흐름
POP terminal boots, displays login screen with line + scheduled shift.
POP 부팅 시 로그인 화면 + 라인·교대 표시.
Operator scans employee badge (or types PIN).
작업자가 배지 스캔(또는 PIN 입력).
System validates via SYS auth: employee active, line authorized, shift matches roster.
SYS 인증 검증: 재직·라인 권한·교대 일치.
Session created (JWT, 12h max). Audio chirp + ✓ confirms.
세션 생성 (JWT, 12시간). chirp + ✓ 확정.
Auto-redirect to IMG-02 Dashboard. Operator name + line + shift in sidebar.
IMG-02 자동 이동. 사이드바에 작업자명·라인·교대 표시.
