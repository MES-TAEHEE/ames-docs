01Overview개요
EN · Description The first quality gate of the A-MES. Automatically triggered when WH-02 PDA Inbound confirms receipt of raw material/fabric. The inspection standard for the item is loaded from MD-08 InspStd (sampling method, AQL level, inspection items, acceptance criteria). Inspector enters measurement results (web) or scans samples (PDA). PASS → stock reflected in WH-03 (status changes from Quarantine to Available). FAIL → stays in WH Quarantine zone, auto-generates NCR (QC-04), and notifies purchasing.
KO · 설명 A-MES 첫 번째 품질 게이트. WH-02 PDA 입고 확정 시 자동 발행. 품목별 검사기준이 MD-08 InspStd에서 자동 로드 (샘플링 방법·AQL 등급·검사 항목·합격 기준). 검사원이 측정값 입력(Web) 또는 샘플 스캔(PDA). 합격 → WH-03 재고 반영(Quarantine → Available 상태 전환). 불합격 → WH 격리구역 잔류, NCR(QC-04) 자동 발행, 구매팀 통보.

03Auto Trigger & Outcome Flow자동 트리거 + 결과 연계
EventTrigger / Action
Trigger INWH-02 PDA Inbound 확정 시 QC-01 자동 발행. 해당 PO/LOT의 MD-08 검사기준 자동 매칭, 샘플 수 자동 계산.
PASS(1) WH-03 재고 Status: Quarantine → Available, (2) WH-03 가용재고 +RcvQty, (3) PP/SCM 발주 대기 해제, (4) Supplier 평가 PASS 1점 가산.
MARGINALQC Lead 검토 대기. 12h 내 미결정 시 자동 FAIL 처리 (BR-QC-04).
FAIL(1) WH 격리위치 (Q-zone) 잔류, (2) NCR(QC-04) 자동 발행, (3) 구매팀 이메일 알림, (4) Supplier Hold ON (다음 입고 시 자동 강화검사 - BR-QC-09), (5) Supplier 평가 FAIL 1점 차감.
3회 FAIL30일 내 동일 Supplier + 동일 Item FAIL 3회 누적 시 CAPA(QC-06) 자동 권고 (BR-QC-03).
04Workflow작업 흐름
WH-02 PDA inbound confirmation auto-triggers QC-01 with MD-08 std auto-matched.
WH-02 PDA 입고 확정 시 QC-01 자동 발행 + MD-08 기준 자동 매칭.
QC Inspector picks up the queue, opens inspection job on Web (or PDA for scan-based AQL).
QC 검사원이 대기열에서 검사 건 선택, Web(또는 PDA)으로 검사 시작.
Pulls calculated samples from the quarantine zone (Q-zone), measures each per the standard.
격리구역(Q-zone)에서 계산된 샘플 수만큼 꺼내어 기준에 따라 측정.
Enters measured values; system auto-judges PASS/FAIL per item. ≥1 FAIL → batch FAIL.
측정값 입력 시 시스템이 항목별 합/부 자동 판정. 1건이라도 FAIL이면 LOT 전체 FAIL.
MARGINAL items routed to QC Lead for final review; FAIL items auto-cascade to NCR (QC-04).
MARGINAL 항목은 QC Lead 재검 routing, FAIL 항목은 NCR(QC-04) 자동 연계.
Final verdict locks the inspection record; downstream updates (WH/Supplier/Buyer) fire within 1s.
최종 판정 시 검사 기록 잠금, 하류 업데이트(WH/Supplier/구매)가 1초 내 발사.
05Business Rules업무 규칙
RuleDescriptionAction
BR-QC-01
Only QC-passed LOTs can be moved from Q-zone to Available stock; FAIL LOTs remain in quarantine.
합격 LOT만 Available 재고로 이관, FAIL 격리구역 잔류.
Block
BR-QC-04
MARGINAL verdict requires QC Lead approval within 12h; otherwise auto-converted to FAIL.
MARGINAL은 QC Lead 12h 내 미결정 시 FAIL로 자동 전환.
Escalate
BR-QC-09
FAIL incoming triggers Supplier Hold ON; next inbound from same supplier auto-routes to 100% inspection.
FAIL 시 Supplier Hold ON + 다음 입고 자동 100% 강화검사.
Auto
BR-QC-03
3 FAILs (same supplier + same item) within 30 days → auto-recommend CAPA (QC-06) and notify QC Manager.
30일 내 동일 공급사·품목 FAIL 3회 → CAPA 자동 권고 + 관리자 알림.
Alert
BR-QC-08
All inspection records (including measurements and photos) retained ≥ 5 years.
검사 기록·측정값·사진 5년 이상 보관 (IATF/감사 대응).
Retain
