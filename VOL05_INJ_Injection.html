INJECTION
MOLDING POP
사출 생산 POP
Shop-floor Point of Production (POP) screens for injection molding lines. Dark-mode UI optimized for factory lighting conditions. Numeric keypad entry for gloved-hand operation. 1-screen-1-action UX principle. All feedback (visual + sound) delivered within 0.3 seconds. Covers WO confirmation, production entry, defect registration, mold change, and Andon.
사출 라인 현장 POP 화면. 공장 조도 대응 다크모드 전용. 장갑 착용 대응 숫자패드 중심. 1화면 1액션 원칙. 0.3초 내 시각+음향 피드백. WO 확인·실적 입력·불량 등록·금형 교체·Andon.

INJ-04
Production Results Entry
생산 실적 입력
🏭 POP Dark1st Dev
EN · Description
Operator enters good-quantity produced per cycle using a large numeric keypad. Good quantity is auto-calculated as (Total − Defects) when defect entry follows. One-tap confirm updates WO progress, auto-increments accumulated shot count on the mounted mold, and feeds OEE calculation. [CT] key inputs the standard cycle quantity.
KO · 설명
작업자가 대형 숫자패드로 사이클당 양품 수량 입력. 불량 입력 시 양품 = 총생산 - 불량 자동 계산. 원터치 확정 → WO 진도 업데이트 + 장착 금형 쇼트수 자동 갱신 + OEE 계산 원천 데이터. [CT] 키 = 표준 사이클 수량.

INJ-01
POP Login
POP 로그인
🏭 POP1st
EN
Operator login via employee badge barcode scan or 4-digit PIN. Line and shift auto-populated from roster. Session expires on shift end or manual logout.
KO
사번 바코드 스캔 또는 4자리 PIN 로그인. 라인·교대 자동 설정. 교대 종료 또는 수동 로그아웃 시 세션 만료.
INJ-02
POP Dashboard
현황판
🏭 POP1st
EN
Dark-mode real-time dashboard: current WO progress bar, equipment status, hourly output chart, defect rate gauge, and mold remaining life. Large font for 5m viewing distance.
KO
다크모드 실시간 현황판: WO 진도바, 설비 상태, 시간별 생산 차트, 불량율 게이지, 금형 잔여 수명. 5m 거리 가독 대형 폰트.
INJ-05
Defect Register
불량 등록
🏭 POP1st
EN
One-tap defect type selection from MD-14 list (6 INJ types). Enter defect quantity via numpad. Defect rate auto-calculated. If rate exceeds 5%, Andon alarm triggers automatically.
KO
MD-14 불량 유형 원터치 선택(INJ 6종). 숫자패드 불량 수량 입력. 불량율 자동 계산. 5% 초과 시 Andon 자동 트리거.
INJ-06
Mold Change Request
금형 교체 요청
🏭 POP2nd
EN
When accumulated shot count reaches 90% of rated life, system warns operator on POP. At 100%, forced mold change request is sent to maintenance team. Mold change downtime auto-recorded for OEE.
KO
쇼트수 90% 도달 시 POP 경고. 100% 도달 시 MNT팀 교체 요청 자동 발송. 금형 교체 비가동 시간 자동 기록.
INJ-08
Andon — Emergency Signal
Andon · 비상 신호
🏭 POP1st
EN
Full-screen red Andon alert on POP when defect rate exceeds 5% or mold life reaches 100%. Simultaneously: flashes line stack light (red), sends push notification to supervisor, creates MNT stop record. Operator must acknowledge to resume production. All Andon events logged with timestamp and reason.
KO
불량율 5% 초과 또는 금형 수명 100% 도달 시 POP 전체 화면 빨간 Andon 알람. 동시에: 라인 스택라이트 빨간 점멸, 관리자 Push 알림, MNT 정지 기록 생성. 작업자 확인 후 재가동. 전체 Andon 이력 타임스탬프 기록.
