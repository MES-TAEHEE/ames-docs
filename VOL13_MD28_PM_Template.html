01Overview개요
EN · Description MD_PMTemplate is the preventive-maintenance task template catalog — a new master added for the MNT module. Each template applies to an equipment type and defines the checklist (JSON), the cycle basis (CALENDAR / RUNTIME / COUNTER), the interval, the required spare parts, the standard duration and whether LOTO (Lock-Out Tag-Out) safety is required. MNT-05 generates a PM schedule when a template's interval is reached; MNT-07 creates the PM Work Order from the template's checklist.
KO · 설명 MD_PMTemplate은 예방정비 작업 템플릿 카탈로그로 — MNT 모듈을 위해 추가된 신규 마스터다. 각 템플릿은 설비 유형에 적용되며 체크리스트(JSON), 주기 기준(캘린더/가동시간/카운터), 간격, 소요 정비자재, 표준 시간, LOTO(잠금·표지) 안전 필요 여부를 정의한다. MNT-05는 템플릿 간격 도래 시 PM 일정을 생성하고, MNT-07은 템플릿 체크리스트로 PM 작업지시를 생성한다.

03Constraints & Index제약조건 + 인덱스
TypeNameDefinition · 정의
PKPK_MD_PMTemplatePRIMARY KEY (PMTemplateID)
CKCK_PM_BasisCHECK (CycleBasis IN ('CALENDAR','RUNTIME','COUNTER'))
CKCK_PM_UnitCHECK (IntervalUnit IN ('DAY','HOUR','SHOT','COUNT'))
CKCK_PM_IntervalCHECK (IntervalValue > 0)
CKCK_PM_DurationCHECK (StdDurationMin > 0)
CKCK_PM_BasisUnitCHECK ((CycleBasis='CALENDAR' AND IntervalUnit='DAY') OR (CycleBasis='RUNTIME' AND IntervalUnit='HOUR') OR (CycleBasis='COUNTER' AND IntervalUnit IN ('SHOT','COUNT')))
IXIX_PM_EquipTypeINDEX (EquipType, ActiveFlag)
⚙ Cycle Basis ↔ Unit · 주기 기준과 단위CK_PM_BasisUnit이 정합을 강제한다 — CALENDAR는 DAY, RUNTIME은 HOUR, COUNTER는 SHOT/COUNT만 허용. 금형 PM은 COUNTER+SHOT, 일반 설비는 CALENDAR+DAY 또는 RUNTIME+HOUR. · The basis and unit must match.

05Validation & Business Rules검증 + 업무 규칙
RuleDescriptionAction
BR-MD-21
A PM template must reference a valid EquipType; MNT-05 cannot schedule a PM without a template.
PM 템플릿 없으면 MNT-05 PM 일정 불가.
Validation
VR-PM-01
CycleBasis and IntervalUnit must be consistent (CK_PM_BasisUnit).
주기 기준과 간격 단위는 정합 (캘린더-일, 가동시간-시간, 카운터-쇼트/카운트).
Validation
VR-PM-02
RequiredPartsJSON entries must reference valid PartNos in MD_SparePart.
소요 자재 JSON은 MD_SparePart 유효 부품만 참조.
FK
BR-MD-23
A template referenced by a PM schedule cannot be deleted — set ActiveFlag = FALSE.
PM 일정 참조 템플릿은 삭제 불가 — 비활성화만.
Block
BR-MD-24
PM template create / update requires an approval workflow + SYS-05 audit log.
PM 템플릿 생성·수정은 승인 워크플로 + 감사 로그.
Audit
