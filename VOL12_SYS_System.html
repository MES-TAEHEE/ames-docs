SYSTEM
ADMINISTRATION
시스템 관리 — 계정 · 권한 · 설정 · 모니터 · 감사
The administration backbone of A-MES. User account management (SYS-01) with auto-lock security. RBAC permission matrix (SYS-02) — the core, governing 7 roles' access to every module. Factory calendar (SYS-03) feeding PP capacity. System configuration (SYS-07) for global parameters and common codes. Notification settings (SYS-06) for 8 event types across 3 channels. Interface monitor (SYS-04) watching SCM/ERP/PLC links. 5-year immutable audit log (SYS-05) capturing every action. System health dashboard (SYS-08) monitoring servers, resources, backups, and jobs. All screens are Admin / System Manager only.
A-MES의 관리 중추. 사용자 계정 관리(SYS-01) 자동 잠금 보안. RBAC 권한 매트릭스(SYS-02) — 핵심, 7역할의 전 모듈 접근 통제. 공장 캘린더(SYS-03) PP 능력 산정 기반. 시스템 설정(SYS-07) 전역 파라미터·공통 코드. 알림 설정(SYS-06) 8종 이벤트 3채널. 인터페이스 모니터(SYS-04) SCM·ERP·PLC 연동 감시. 5년 위변조 불가 감사 로그(SYS-05) 전 행위 기록. 시스템 헬스 대시보드(SYS-08) 서버·자원·백업·잡 감시. 전 화면 Admin·시스템 관리자 전용.
