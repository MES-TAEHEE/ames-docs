VOL.12 · SYS · System Administration · 시스템 관리
SYSTEM
ADMINISTRATION
시스템 관리
System administration module for A-MES. Covers user account management, RBAC (Role-Based Access Control) with 7 roles, factory calendar configuration, SCM/ERP interface monitoring, 5-year audit log, and configurable notification settings for 8 event types. All screens restricted to Admin and System Manager roles.
A-MES 시스템 관리 모듈. 사용자 계정 관리, RBAC 7역할 권한 관리, 공장 캘린더 설정, SCM·ERP 인터페이스 모니터링, 5년 감사 로그, 8종 이벤트 알림 설정. 전체 화면 Admin·시스템 관리자 역할 한정.

SYS-01
User Account Management
사용자 계정 관리
🖥 Web1st
EN
Create, edit, and deactivate user accounts. Each account: employee ID, name, department, role assignment (from 7 RBAC roles), assigned lines (for POP users), and active status. Password reset by Admin only. 5 consecutive login failures → account auto-locked. Locked accounts shown in red. Email notification to user and Admin on lock.
KO
사용자 계정 생성·수정·비활성화. 각 계정: 사번·이름·부서·역할(7종 RBAC)·담당 라인(POP 사용자)·활성 상태. 비밀번호 초기화는 Admin만 가능. 로그인 5회 연속 실패 → 계정 자동 잠금. 잠금 계정 빨간색 표시. 사용자·Admin 이메일 알림.
