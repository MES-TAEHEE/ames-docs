01Overview개요
EN · Description MD_CommonCode is the system-wide key-value dictionary that backs every dropdown and enumeration in A-MES. Each row belongs to a code group (e.g. PROCESS_CODE, UOM_CLASS, SHIFT) and pairs a code value with a bilingual name. A self-referencing parent allows two-level groups, a sort order fixes display sequence, and a generic attribute column carries group-specific extra data. This table is loaded first in Phase-0 because nearly every other master references its enumerations.
KO · 설명 MD_CommonCode는 A-MES의 모든 드롭다운·열거형을 지탱하는 전사 키-값 사전이다. 각 행은 코드 그룹(예: PROCESS_CODE, UOM_CLASS, SHIFT)에 속하며 코드값과 한·영 명칭을 짝짓는다. 자기참조 부모로 2단계 그룹을 구성하고, 정렬 순서로 표시 순번을 고정하며, 범용 속성 컬럼이 그룹별 부가 데이터를 보유한다. 거의 모든 마스터가 이 열거형을 참조하므로 Phase-0에서 최우선 적재된다.
