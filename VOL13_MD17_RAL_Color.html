01Overview개요
EN · Description MD_RALColor registers the RAL color standards used on the powder-painting line. Each row pairs a RAL code with its painting recipe — cure temperature, cure duration, electrostatic voltage and target particle size — and points at the current powder lot in use. A customer-mapping JSON resolves each OEM's part-color codes to the shared RAL standard. PNT-04 reads this table to drive oven set-points; an inactive color cannot be scheduled.
KO · 설명 MD_RALColor는 분체도장 라인에서 사용하는 RAL 컬러 표준을 등록한다. 각 행은 RAL 코드와 도장 레시피 — 경화 온도, 경화 시간, 정전 전압, 목표 입자 크기 — 를 짝지으며 사용 중인 현재 분체 로트를 가리킨다. 고객 매핑 JSON이 각 OEM의 부품 컬러 코드를 공유 RAL 표준으로 변환한다. PNT-04가 이 테이블을 읽어 오븐 설정값을 구동하며, 비활성 컬러는 스케줄링 불가하다.

05Validation & Business Rules검증 + 업무 규칙

BR-MD-13
PNT-04 derives oven and gun set-points from the RAL recipe fields.
PNT-04가 RAL 레시피 필드에서 오븐·도장건 설정값을 도출.
Auto
BR-MD-14
An ActiveFlag=FALSE color cannot be scheduled on a painting order.
ActiveFlag=FALSE 컬러는 도장 작업지시 스케줄링 불가.
Block
VR-RAL-01
CureTemp must stay within the powder spec band (140–220°C).
경화 온도는 분체 사양 대역(140~220°C) 내 유지.
Block
VR-RAL-02
If CurrentPowderLot is EXPIRED, the painting order is blocked until re-assigned.
현재 분체 로트가 EXPIRED면 재지정 전까지 도장 작업지시 차단.
Block
VR-RAL-03
CustomerMapJSON keys must resolve to a registered RALCode.
고객 매핑 JSON 키는 등록된 RALCode로 변환되어야 함.
Validate
