01Overview개요
EN · Description Real-time finished goods inventory view. Stock segmented by Status: Available (ready to ship), Reserved (linked to open FG-03 Shipment Order), Hold (QC-05 quarantine blocked). Visual zone heatmap + item-level table + customer-aged inventory. Reservation auto-locks Available qty when FG-03 confirms; release on FG-04 Picking complete or order cancel. Read-only access for Sales (customer-scoped) and QC (Hold scope).
KO · 설명 실시간 완제품 재고. Status 분리: Available(출하 가능), Reserved(FG-03 출하지시 연결), Hold(QC-05 격리 차단). 위치 zone 히트맵 + 품목 테이블 + 고객별 체류일수. FG-03 확정 시 Available 자동 Reserved 락, FG-04 피킹 완료 또는 주문 취소 시 해제. Sales(고객 범위)·QC(Hold 범위) 읽기 전용.

02Stock Status Definitions재고 상태 정의
StatusDisplay ColorDefinition · Trigger
AVAILABLEGreen출하 가능 상태. FG-01 Confirm 시 진입. 어떤 고객 주문에도 할당 가능.
RESERVEDPurpleFG-03 Shipment Order 확정 시 자동 락. 해당 주문 외에 할당 불가. FG-04 Pick 완료 시 해제 (SHIPPED 전환).
HOLDRedQC-05 격리 활성. 모든 출하/이동 차단 (DB constraint). NCR/Disposition 완료 후 해제.
SHIPPEDBlueFG-05 Loading Confirm 후 출하 처리됨. 재고에서 차감, FG-08 이력 보관.
RETURNEDYellowFG-RTN 반품 처리. 재 QC 검사 후 RESTOCK / SCRAP / CLAIM 분기.
03Refresh & Reservation갱신 + 예약 메커니즘
EventAction
FG-01 StockingWebSocket: FG_Stock INSERT → 0.5s 내 화면 즉시 반영 (KPI + Heatmap + Item table).
FG-03 Order ConfirmAvailable Qty → Reserved 자동 락 (Atomic UPDATE). Optimistic locking으로 동시성 충돌 방지.
FG-04 Pick CompleteReserved Qty → SHIPPED (재고 -). Live broadcast.
QC-05 HoldAvailable/Reserved Qty → HOLD 강제 전환 (NCR 발생). Reserved 였다면 해당 FG-03 주문에 alert.
Order CancelReserved Qty → Available 즉시 복구. 30분 무활동 자동 해제도 가능 (옵션).
04Workflow작업 흐름
FG Manager opens dashboard; sees KPI tiles + heatmap at-a-glance for full-zone status.
FG Manager가 대시보드 오픈, KPI + 히트맵으로 전체 zone 상황 즉시 파악.
Filters by Customer / Item / Aged to narrow focus; aging colors highlight problem stock.
고객/품목/체류일수 필터로 좁히기, 체류 색상으로 문제 재고 강조.
Drill-down by clicking slot → location detail page (WO/LOT/Pallet list).
슬롯 클릭 → 위치 상세 (WO/LOT/팔레트 목록).
Sales user accesses customer-scoped view only; cannot see other customers' data.
Sales 사용자는 자기 고객 데이터만 접근, 타 고객 비공개.
QC user reviews Hold items and drills to QC-05 for Disposition planning.
QC 사용자는 Hold 항목 검토, QC-05로 드릴해 처분 계획.
Export Excel for monthly reporting; daily snapshot stored for FG-08 history.
월간 보고용 Excel 출력, 일일 스냅샷은 FG-08 이력 보관.
05Business Rules업무 규칙
RuleDescriptionAction
BR-FG-06
Reservation is atomic: cannot reserve qty exceeding Available; race conditions blocked with optimistic locking.
예약 원자성: Available 초과 예약 불가, 동시성 충돌 차단.
Block
BR-FG-07
HOLD status overrides RESERVED: existing reservations on held items are alert + auto-released.
HOLD가 RESERVED 우선, 격리 시 기존 예약 알림 + 자동 해제.
Force
BR-FG-08
Sales role sees only their assigned customer's stock; other customers hidden at SQL layer.
Sales는 담당 고객 재고만, 타 고객 SQL 레이어에서 차단.
Scope
BR-FG-09
Aged stock ≥14d highlights yellow, ≥30d red. Auto-notify FG Manager on 30d+ items.
14일 이상 노랑, 30일 이상 빨강 + FG Manager 알림.
Alert
BR-FG-10
Daily snapshot of inventory state stored for FG-08 history and audit.
일일 재고 스냅샷 FG-08 이력·감사 보관.
Auto
