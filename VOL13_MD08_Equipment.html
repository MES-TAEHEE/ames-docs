01Overview개요
EN · Description MD_Equipment registers every production machine — injection machines, wrapping presses, painting robots, spray booths. Each row carries its line / Work Center assignment, the theoretical cycle time and OEE target that feed RPT-06, a JSON list of compatible molds, the PLC signal address for real-time status, and the live 4-state status (RUN / IDLE / DOWN / PM). It is the backbone of the MNT module — MNT-01~09 all key off EquipID.
KO · 설명 MD_Equipment는 모든 생산 설비를 등록한다 — 사출기·감싸기 프레스·도장 로봇·도장부스. 각 행은 라인/작업장 배정, RPT-06에 공급되는 이론 사이클·OEE 목표, 호환 금형 JSON 목록, 실시간 상태 PLC 신호 주소, 4-상태 현재 상태(RUN/IDLE/DOWN/PM)를 보유한다. MNT 모듈의 중추로 — MNT-01~09가 모두 EquipID를 기준으로 동작한다.

05Validation & Business Rules검증 + 업무 규칙
RuleDescriptionAction
VR-Equip-01
Status is updated by the PLC feed (MNT-01), not manually — manual edit is restricted to Admin.
상태는 PLC 신호(MNT-01)가 갱신 — 수동 편집은 Admin 한정.
Restrict
VR-Equip-02
An equipment's EquipType must be consistent with its Work Center ProcessType.
설비 유형은 작업장 공정 유형과 정합.
Validation
VR-Equip-03
MoldCompatJSON entries must reference valid MoldIDs in MD_Mold.
호환 금형 JSON은 MD_Mold 유효 금형만 참조.
FK
BR-MD-23
An equipment in use cannot be deleted — set ActiveFlag = FALSE (retired).
사용 중 설비는 삭제 불가 — ActiveFlag=FALSE 처리.
Block
BR-MD-24
Equipment create / update requires an approval workflow + SYS-05 audit log.
설비 생성·수정은 승인 워크플로 + SYS-05 감사 로그.
Audit
