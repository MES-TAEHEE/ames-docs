MASTER DATA MANAGEMENT
기준정보 관리
Foundation of the entire A-MES system. 14 master data entities must be entered in Phase 0 before any operational screen functions. Key masters: Item, BOM, BOP Routing, Vendor (SAV/GEO), Equipment, Location (WH-08), Mold, Inspection Standard, Defect Types.
A-MES 전체 시스템의 기반. 14개 기준정보를 Phase 0에서 아래 순서로 입력 완료해야 운영 화면이 동작. 핵심 마스터: 품목, BOM, BOP 라우팅, 거래선(SAV/GEO), 설비, 로케이션(WH-08), 금형, 검사기준, 불량유형.

SCREEN FEATURES · 주요 기능
📋
MD-01 Item Master
품목 마스터
All items: raw material, fabric, paint, semi-finished, finished goods. Item code, name, type, unit, min/max stock.
전 품목: 원자재·원단·도료·반제품·완제품. 품목코드·명·유형·단위·최소/최대재고.
🌲
MD-02 BOM — Bill of Materials
자재명세서
Multi-level BOM per finished good. Component item, qty, unit. Links to MRP. Version-controlled (MD-03).
완제품별 다단계 BOM. 구성 품목·수량·단위. MRP 연계. 버전 관리(MD-03).
🔀
MD-04 BOP — Bill of Process
공정 경로
Routing per product group. Door Trim/Console: INJ→IMG→QC→FG. Interior Painted Parts (TBD): INJ→PNT→QC→FG.
제품군별 공정 경로. 도어트림/콘솔: INJ→IMG→QC→FG. 내장 도색 사출품(TBD): INJ→PNT→QC→FG.
🏢
MD-07 Vendor Master
거래선 마스터
Vendor code, name, type (material supplier / customer SAV-GEO), contact, EDI/SCM link. SAV & GEO as ship-to customers.
거래선 코드·명·유형(공급사/고객 SAV-GEO)·연락처·EDI/SCM 연결. SAV·GEO 납품 거래선.
⚙️
MD-08 Equipment Master
설비 마스터
Equipment ID, name, line, type (injection/painting/wrapping), capacity, mold compatibility, current status.
설비ID·명·라인·유형(사출/도장/감싸기)·생산능력·금형 호환성·현재 상태.
🔬
MD-06 Inspection Standard
검사 기준
Inspection items, measurement method, tolerance, AQL level per item/process. Source for QC-01/02/03 checklists.
검사 항목·측정 방법·허용 공차·AQL 레벨. QC-01/02/03 체크리스트 원본.
📐
MD-12 Defect Types
불량 유형 마스터
All defect codes per module (INJ-D01~06, IMG-D01~06, PNT-D01~06, QC). Linked to CAPA trigger rules.
모듈별 전체 불량 코드(INJ-D01~06, IMG-D01~06, PNT-D01~06, QC). CAPA 트리거 연계.
📦
MD-11 Location Master
로케이션 마스터
WH zone/row/column/level hierarchy. QR label generation. Capacity per location. (= WH-08 screen)
WH 구역/행/열/층 계층 구조. QR 라벨 생성. 로케이션별 용량. (= WH-08 화면)
