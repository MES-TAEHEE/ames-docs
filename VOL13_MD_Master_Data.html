기준정보 관리 · 가동 전 전원 입력 필수
Foundation data for all A-MES modules. Must be fully entered before go-live (Phase 0). Mandatory input order: ① Vendor → ② Item → ③ BOM → ④ BOP Routing → ⑤ Equipment → ⑥ Fabric/Paint → ⑦ Inspection Standard → ⑧ Defect Codes → ⑨ Stock Levels → ⑩ Mold/PM Standards → ⑪ Locations. Total 14 master data screens. No production transactions can be created without all required masters.
전체 A-MES 모듈의 기반 데이터. 가동 전 Phase 0에 전원 입력 필수. 필수 입력 순서: ①거래선→②품목→③BOM→④BOP라우팅→⑤설비→⑥원단/도료→⑦검사기준→⑧불량유형→⑨재고기준→⑩금형/PM→⑪로케이션. 총 14개 마스터 데이터 화면. 필수 마스터 미입력 시 생산 트랜잭션 생성 불가.

MD-14
Defect Type Master (18 Types)
불량 유형 마스터 (18종)
🖥 WebPhase 0
EN
18 standardized defect codes covering all production processes (INJ 6 types, IMG 6 types, PNT 6 types). Each code: defect ID, name (EN/KO), applicable process, severity (Minor/Major/Critical), and standard photo example. Used by POP defect register (INJ-05, IMG-05, PNT-04) and QC defect analysis (QC-05, RPT-02).
KO
전체 공정 18종 표준 불량 코드(INJ 6종, IMG 6종, PNT 6종). 각 코드: 불량ID·명칭(영/한)·적용공정·심각도(경미/주요/치명)·표준 사진 예시. POP 불량 등록(INJ-05, IMG-05, PNT-04) + QC 불량 분석(QC-05, RPT-02) 연계.

MD-06
Vendor Master (SAV / GEO)
거래선 마스터
🖥 WebPhase 0
EN
Registers client plants (SAV – Savannah, GEO – Georgia) and raw material suppliers. Each vendor: code, name, address, contact, delivery note format (SAV/GEO specific), and EDI/SCM connection settings. Required for PO creation, SCM sync, and FG-06 delivery note formatting.
KO
거래선(SAV-사바나, GEO-조지아) + 원자재 공급사 등록. 코드·명칭·주소·담당자·거래명세서 양식(SAV/GEO 전용)·SCM 연결 설정. PO 생성·SCM 동기화·FG-06 거래명세서 생성 기준.
MD-05
Equipment Master
설비 마스터
🖥 WebPhase 0
EN
Registers all production equipment: code, name, type (Injection/Wrapping/Paint/Oven/Other), rated capacity (cycle time, daily capacity), assigned line, and installation date. Used by CRP capacity planning and MNT-01 equipment card.
KO
전체 생산 설비 등록: 코드·명칭·유형(사출/감싸기/도장/오븐/기타)·정격 능력(사이클타임·일 생산능력)·담당 라인·설치일. CRP 능력 계획·MNT-01 설비 카드 기준.
MD-13
Inspection Standard
검사 기준
🖥 WebPhase 0
EN
Defines inspection checklist items, measurement methods, acceptance criteria, and sampling plan (AQL level or 100%) per item and inspection type (Incoming / In-Process / Final). Auto-loaded into QC-01/02/03 inspection screens.
KO
품목별·검사 유형(수입/공정/최종)별 검사 체크리스트·측정 방법·합부 기준·샘플링 계획(AQL/전수) 정의. QC-01/02/03 검사 화면에 자동 로드.
MD-09/10/11/12
Stock Levels / Mold / PM Standards
재고기준·금형·PM 기준
🖥 WebPhase 0
EN
MD-09: Min/max stock levels per item (WH alert threshold). MD-10: Paint color recipe definitions (PNT-02 color setup). MD-11: Mold master — rated shot life, current count. MD-12: PM standard — interval, task list, required parts per equipment.
KO
MD-09: 품목별 최소·최대 재고(WH 알림 기준). MD-10: 도료 색상 레시피(PNT-02 색상 세팅). MD-11: 금형 마스터 — 정격 쇼트 수명·현재 카운트. MD-12: PM 기준 — 설비별 주기·작업목록·필요부품.
