01Overview개요
EN · Description MD_Mold registers injection molds with their shot-count life management. Each row holds the rated shot life and the cumulative current shots; the system raises a warning at 90% and a forced production block at 100% of rated shots, auto-creating a MOLD_CHANGE Work Order in MNT-07. It also carries cavity count, required tonnage and a JSON list of compatible items. CurrentShots is auto-incremented by INJ production transactions — it is not edited manually.
KO · 설명 MD_Mold은 사출 금형을 쇼트수 수명 관리와 함께 등록한다. 각 행은 정격 쇼트 수명과 누적 현재 쇼트수를 보유한다 — 시스템은 정격 대비 90%에서 경고를, 100%에서 생산 강제 차단을 발생시키고 MNT-07에 MOLD_CHANGE 작업지시를 자동 생성한다. 캐비티 수, 필요 형체력, 호환 품목 JSON 목록도 보유한다. CurrentShots는 INJ 생산 트랜잭션이 자동 증가시키며 수동 편집하지 않는다.

03Constraints & Index제약조건 + 인덱스
TypeNameDefinition · 정의
PKPK_MD_MoldPRIMARY KEY (MoldID)
CKCK_Mold_StatusCHECK (Status IN ('ACTIVE','MAINT','RETIRED'))
CKCK_Mold_RatedCHECK (RatedShots > 0)
CKCK_Mold_ShotsCHECK (CurrentShots >= 0)
CKCK_Mold_CavityCHECK (CavityCount > 0)
IXIX_Mold_StatusINDEX (Status)
IXIX_Mold_LifeINDEX ((CurrentShots * 100 / RatedShots)) — 수명 임계 모니터링 (계산 인덱스).
⚙ Shot-Life Threshold · 쇼트수 임계수명율 = CurrentShots ÷ RatedShots × 100. ≥90% → MNT 경고 알림 · ≥100% → INJ 생산 강제 차단 + MNT-07 MOLD_CHANGE WO 자동 생성. 금형 교체·재생 시 CurrentShots는 0으로 초기화 (감사 이력 보존). · 100% triggers a forced block and an auto MOLD_CHANGE WO.

05Validation & Business Rules검증 + 업무 규칙
RuleDescriptionAction
VR-Mold-01
CurrentShots is auto-incremented by INJ production; manual edit is restricted to mold-change/refurbish reset.
CurrentShots는 INJ 생산이 자동 증가 — 수동 편집은 교체·재생 초기화만.
Restrict
VR-Mold-02
At ≥ 90% life the system warns; at ≥ 100% INJ production is blocked + a MOLD_CHANGE WO is auto-created.
수명 90% 경고, 100% INJ 차단 + MNT-07 교체 WO 자동.
Block
VR-Mold-03
CompatItemsJSON entries must reference valid ItemNos in MD_Item.
호환 품목 JSON은 MD_Item 유효 품목만 참조.
FK
BR-MD-23
A mold in use cannot be deleted — set Status = RETIRED.
사용 중 금형은 삭제 불가 — Status=RETIRED 전환만.
Block
BR-MD-24
Mold create / update / shot-reset requires an approval workflow + SYS-05 audit log.
금형 생성·수정·쇼트 초기화는 승인 워크플로 + 감사 로그.
Audit
