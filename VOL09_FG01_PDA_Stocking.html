01Overview개요
EN · Description The entry point for finished goods. Auto-triggered the instant QC-03 issues a PASS verdict (FGTriggerID from QC-03). FG Operator picks up the trigger on PDA, scans the WO barcode, picks an empty FG location (suggested by Slotting algorithm), scans the location QR, prints & affixes a pallet label, and confirms. Inventory is immediately reflected in FG-02 with Status=Available. Customer-grouped slotting keeps shipments ready for fast picking.
KO · 설명 완제품 진입 화면. QC-03 PASS 발행 즉시 자동 트리거 (FGTriggerID 수신). FG 작업자가 PDA에서 트리거 인수, WO 바코드 스캔, 빈 FG 위치 선택(Slotting 알고리즘 추천), 위치 QR 스캔, 팔레트 라벨 인쇄·부착, 확정. FG-02 재고에 Status=Available로 즉시 반영. 고객별 구역 슬로팅으로 빠른 피킹 준비.

03Auto Trigger Flow자동 트리거 흐름
SourceTrigger Action
QC-03 PASSQC-03 최종검사 PASS 확정 즉시 FG-01 큐에 작업 push (FGTriggerID 발급), FG Operator PDA에 알림.
FG-01 Slotting슬로팅 알고리즘 자동 호출: (1) 고객 zone 우선, (2) 임박 ship-date 슬롯 우선, (3) FIFO 호환성, (4) 가용 capacity.
FG-01 Confirm(1) FG_Stock INSERT (Status=Available), (2) WH 인접 PASS 통보, (3) 팔레트 라벨 자동 인쇄, (4) QC-03 record에 StockID 역참조, (5) RPT 모듈에 입고 데이터 적재.
Pickup Date 임박Ship-date D-2부터 FG-02에서 highlighted, FG-03 Shipment Order 후보로 자동 표시.
04Workflow작업 흐름
QC-03 issues PASS → FGTriggerID auto-pushed to FG Operator PDA queue within 1s.
QC-03 PASS 발행 → 1초 내 FG 작업자 PDA 큐에 자동 push.
Operator scans WO barcode on the printed pickslip; system loads WO/Item/Qty/Customer.
작업자가 인쇄된 픽슬립의 WO 바코드 스캔, 시스템이 WO/품목/수량/고객 자동 로드.
System auto-suggests location via slotting algorithm; operator can scan suggested QR or override.
시스템이 슬로팅 알고리즘으로 위치 추천, 추천 QR 스캔 또는 다른 위치 선택.
Operator scans the chosen location QR; system validates capacity and item-mix rules.
선택 위치 QR 스캔, 시스템이 capacity + 품목 혼적 규칙 검증.
Label printer auto-issues pallet labels (qty/PackSpec splits); operator affixes them.
라벨러 자동 인쇄 (PackSpec 기준 분할), 작업자가 팔레트에 부착.
Operator taps Confirm → FG_Stock INSERT + FG-02 instant refresh; RPT data feed fires.
Confirm 터치 → FG_Stock INSERT, FG-02 즉시 갱신, RPT 데이터 발사.
05Business Rules업무 규칙
RuleDescriptionAction
BR-FG-01
Only QC-03 PASS WOs can be stocked. DB constraint at FG_Stock INSERT.
QC-03 합격 WO만 적재 가능 (DB 제약).
Block
BR-FG-02
Slotting algorithm prioritizes customer zone + ship-date proximity. Override requires reason logged.
슬로팅: 고객 zone + 임박 ship-date 우선. 거부 시 사유 자동 기록.
Auto
BR-FG-03
Pallet labels must be printed and scanned-back to confirm placement; missing scan blocks confirm.
팔레트 라벨 인쇄·재스캔 필수, 누락 시 Confirm 차단.
Block
BR-FG-04
Item-mix rule: Customer-A items cannot share a slot with Customer-B (BR-MD-08 PackSpec).
동일 슬롯에 다른 고객 품목 혼적 불가 (BR-MD-08).
Block
BR-FG-05
Stocking records retained ≥ 5 years (audit + recall).
적재 기록 5년 이상 보관 (감사 + 리콜).
Retain
