01Overview개요
EN · Description Entered from IMG-03 [Defect] button. Operator picks one of 6 IMG-specific defect types (LIFTING / WRINKLE / BOND_BLEED / FAB_SOIL / COLOR_DIFF / CUT_DEFECT) via large tap-buttons, enters quantity. Defect rate auto-calculated; ≥ 5% triggers Andon. Special: COLOR_DIFF defects auto-link to recent FabricInputAttempt for root-cause; FAB_SOIL ≥ 3 within shift triggers handling cleanliness alert.
KO · 설명 IMG-03 [Defect] 버튼에서 진입. 6종 IMG 전용 불량(들뜸/주름/본드번짐/오염/색상/재단) 중 1개를 대형 버튼으로 선택, 수량 입력. 불량률 자동 계산, 5% 이상 시 Andon 발동. 특수 처리: COLOR_DIFF는 최근 FabricInputAttempt와 자동 연계, FAB_SOIL 교대 누적 3건 이상 시 취급 청결도 알림.

03Workflow작업 흐름
IMG-03 [Defect] tapped → IMG-05 opens with cycle EntryID context.
IMG-03 [Defect] 터치 → 사이클 컨텍스트 인계.
Operator picks defect type. Likely cause + action suggestion displayed.
유형 선택. 추정 원인 + 조치 표시.
Enter qty on numpad. Real-time rate preview shown.
수량 입력. 실시간 불량률 미리보기.
Confirm → INSERT IMG_Defect. Auto-link special rules fire (§02 alert).
Confirm → IMG_Defect 적재. 특수 자동 연계 동작.
Rate ≥ 5% → fire Andon (BR-IMG-06). Rate < 5% → return to IMG-03.
5% 이상 시 Andon 발동. 미만 시 IMG-03 복귀.
